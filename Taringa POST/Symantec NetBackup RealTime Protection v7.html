Symantec NetBackup RealTime Protección v7 | 3.32 GB

RealTime NetBackup ofrece una protección continua de datos (CDP) y replicación de las aplicaciones críticas para ayudar a reducir drásticamente el tiempo de recuperación y la pérdida de datos. Ventanas de backup pueden ser eliminados y los administradores pueden recuperar al instante o a cualquier punto en el tiempo. A diferencia de los métodos de backup tradicionales en tiempo real del software CDP puede recuperar terabytes de datos en cuestión de segundos sin ningún tipo de almacenamiento adicional.
Características principales
- Protección de datos continua para cualquier punto en el tiempo de recuperación.
- Recuperación usando volúmenes virtualizados disco llamado "TimeImages".
- Integración con NetBackup para la aplicación consciente de la protección de impacto cero.

Nuevas características
- La protección de datos Continious y cualquier punto en el tiempo de recuperación de VMDK máquinas virtuales con VMware.
- Integración con VMware vCenter.
- Se ha añadido una nueva protección para RedHat Enterprise Linux.
- Se ha añadido una nueva protección para Microsoft Windows Server 2008 R2.
- Se ha añadido una nueva protección para los locales de almacenamiento de conexión directa.
- Nuevo diseño de la instalación y configuración para mayor simplicidad.
- Viene equipada con NetBackup 7.1.
- NetBackup siempre está listo en la ubicación DR.
- Protege el catálogo de la corrupción histórica o lógica.

Protegidos los sistemas operativos
- Microsoft Windows Server 2003, 2008, 2008 R2 (de 32 bits y x64 Edition)
- UNIX - Sun Solaris 9 y 10 (SPARC)
