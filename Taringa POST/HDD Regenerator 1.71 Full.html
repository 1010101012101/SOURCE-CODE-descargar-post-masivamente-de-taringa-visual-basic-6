Què es HDD Regenerator ?

HDD Regenerator es un programa único para la regeneración de discos duros. El programa elimina daños fisicos ( sectores defectuosos ) desde la superficie del disco duro, restaurandolo. HDD Regenerator puede reparar disco duros dañados sin afectar o cambiar la data existente. Algunas de sus principales caracteristicas son:

Habilidad de detectar sectores fisicos defectuosos en la superficie del disco duro.
Habilidad de reparar sectores fisicos dañados ( errores magnéticos ) en la superficie del disco duro.
El programa ignora el sistema de archivo, ya que escanea el disco a nivel fisico. Puede ser usado con FAT, NTFS o cualquier sistema de archivo, y también puedes ser usado con discos duros sin formatear o sin particionar.by tano1221
Puedes iniciar el proceso directamente bajo Windows XP y Vista
Puedes iniciar el proceso bajo DOS creando previamente desde HDD Regenerator a bootable flash o CD/DVD.

Si no puedes bootear tu PC a Windows, instala HDD Regenerator en otra PC con Windows XP/Vista y crea a bootable flash/CD/DVD, luego bootea tu PC desde este flash/CD creado.

Como el programa no cambia la estructura logica del disco duro, el sistema de archivos puede contener marcados sectores defectuosos de analisis previos, incluso si el disco duro ya está regenerado y libre de sectores defectuosos. Si deseas remover esas marcas, reparticiona el disco duro.

Como iniciar HDD Regenerator:

Una vez instalado y activado, para iniciar el programa da doble click en el icono de HDD Regenerator en tu escritorio y puedes iniciar el proceso de reparación de 3 formas:

directamente desde Windows: da click en la pestaña Regeneration | Start Process under Windows, para Windows Vista se necesita tener 2 discos duros para realizar el proceso directamente desde Windows.
crear una unidad flash bootable: da click en Regeneration | Create Bootable Flash"
crear un CD/DVD bootable: da click en Regeneration | Create Bootable Flash

Despues de que el Flash/CD/DVD fue creado, bootea tu PC desde este Flash/CD/DVD y la consola de aplicación comenzará bajo DOS.

Una vez que hayas iniciado el programa por uno de estos 3 medios: by tano1221

Elige el dispositivos a regenerar
Elige el modo ("Scan and Repair" o "Scan Only"("Escanear y Reparar" o "Escanear solamente"
Ingresa el parametro de "Starting Sector" ( Sector de Inicio ). Deja este parametro igual a 0 si deseas escanear todo el disco. Si deseas escanear el disco desde algun punto que no sea el inicio, por ejemplo no deseas escanear los primeros 5 GB, porque ya sabes que no contiene sectores defectuosos, en este caso cambio el valor a 5000 con una 'm' despues o 10240000 sin la 'm' (número de sectores 5000*1024*1024/512=10240000)
Espera mientras el escaneo se completa.

*HDD Regenerator puede reparar sectores defectuosos causados por errores magnéticos en la superficie del disco duro. Si el disco duro tiene roturas fisicas (cabezales dañados, plaac del circuito, etc.), entonces no serán reparados por el programa.

Recomendación:

Si el disco duro ya presenta bastantes sectores defectuosos lo mejor es reemplazarlo o mientras no se pueda comprar uno nuevo, hacer un backup de toda la info vital en un CD o DVD o unidad USB, incluso previo al proceso de regeneración con HDD Regenerator.

Luego de usar HDD Regenerator y evitar que el sistema marque sectores defectuosos cuando ya no los hay es recomendable reparticionar el disco duro, y de paso, desfragmentar el disco duro con alguna otra herramienta como el Tuneup o el programa que suelas usar para la desfragmentación.
