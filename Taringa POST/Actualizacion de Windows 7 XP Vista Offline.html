ACTUALIZACION SIN INTERNET (OFFLINE) DE WINDOWS





Microsoft Windows Update es un servicio inteligente integrado con cualquier sistema operativo Windows. Actualización de Windows basada en su estado de activación de Windows 7 le permite descargar actualizaciones, parches y actualizaciones de forma automática en Windows 7.

Si por alguna razón su Windows Update deja de funcionar o que está trabajando en el equipo no está conectado a Internet, entonces usted podría perder algunos cambios importantes críticos

Para ayudar a la actualización de Windows 7 fuera de línea aquí es WSUS conexión utilidad de actualización. WSUS Offline actualización te permite descargar fácilmente los parches y actualizaciones para Windows y Office sin pasar por todo el procedimiento en el sitio web de Windows Update de Microsoft.

Las modificaciones en la versión 6.9.0

· Optional installation of Internet Explorer 9 on Windows Vista, 7 and Server 2008(R2) (x86/x64) systems
· Language Pack support added for .NET Frameworks 3.5 SP1 and 4
· Installation and update packages for .NET Frameworks 3.5 SP1 and 4 will be validated only once a day if verification is enabled
· Fix: Automatic recall preparation failed for new version 3.1 of Sysinternals’ tool Autologon.exe
· Fix: Download exclude lists for Windows 7 / Server 2008 R2 (x86/x64) updated since Microsoft modified their list of included fixes and updates on March 4th, 2011
· Fix: Download filters missed CAPICOM security update kb931906 again
· Fix: Several obsolete cumulative security updates for Internet Explorer remained
