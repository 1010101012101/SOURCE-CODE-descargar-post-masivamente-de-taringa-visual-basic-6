ChromePlus mejorará la experiencia de usuario del navegador Chrome. ChromePlus tiene todas las funcionalidades que Google Chrome tiene pero ChromePlus añadido algunas funciones útiles, tales como gestos del ratón, arrastre Super pestaña IE en cromo, etc Mientras tanto, ChromePlus es libre, sin limitación de función y se puede utilizar ChromePlus para navegar por Internet con más facilidad.
V1.6.2.0 for Windows (based on Chromium 11.0.696.77)
