Se han optimizado todos los archivos, asi como tambien el registro para tener un sistema completo pero mas rapido y estable que el normal, el apagado e inicio del sistema es mas rapido, se optimizo las conecciones de internet entre varias optimizaciones mas (no intentes copiar el registro a otro sistema para optimizarlo, Evo eXPecial contiene optimizacion de archivos que no se encuentran en el registro).
