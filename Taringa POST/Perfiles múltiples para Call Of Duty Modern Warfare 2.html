Hola taringueros! Aca les dejo un programa para tener perfiles múltiples en Call of Duty: Modern Warfare 2 [CODMW2]

• Primero de todo lo que tienen que hacer es descargar el programa desde acá

• Después tienen que extraer el archivo "MW2 Loader.exe" en la carpeta donde tienen guardado el juego.

• Luego diríjanse a la carpeta del juego y creen un acceso directo al escritorio del archivo que extrajeron anteriormente.

• Cuando ya tengan el acceso directo en el escritorio háganle doble-clic y pongan "NEW PROFILE" [si es que quieren un perfil nuevo] "DELETE" [para borrar el perfil seleccionado], "RENAME" [para renombrar el perfil seleccionado] y "LAUNCH" [para abrir el perfil seleccionado y poder jugar]

~ Notas:

• Esto sólo funciona para la versión PC del juego y hasta donde yo sé sólo cambia su perfil de jugador, que incluye todas las opciones del juego y partidas guardadas.

• Si por alguna razón, tienen que volver a seleccionar un programa o un acceso directo para ejecutar, pueden hacer esto para restablecer el cargador. Para ello es necesario eliminar el archivo profiles.xml encuentra en:

(XP) C: Documents and Settings All Users Datos de programa MW2_Loader

(Vista) C: Datos de programa MW2_Loader

(Win7) C: Datos de programa MW2_Loader

Este programa usa Microsoft .Net Framework 3.5, si no lo tenes, lo podes descargar de acá
