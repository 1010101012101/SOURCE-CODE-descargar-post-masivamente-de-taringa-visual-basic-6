EMPECEMOS

1. Descargar el OllyDbg.

2. Abrir la ExDB con el OllyDbg.

3. Les aparecerá algo como esto:

UPDATE Guild SET G_Notice='%s'

4. Modificar la "s" por una "d" les quedará asi:

UPDATE Guild SET G_Notice='%d'

5. Ya esta, ya tienen su ExDB fixeada y protegida.
