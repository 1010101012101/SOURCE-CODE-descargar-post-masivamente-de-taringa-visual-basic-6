El reproductor de audio y video Windows Media Player es un producto desarrollado por Microsoft, frecuentemente abreviado como WMP. En cada versión de Windows el reproductor WindowsMedia Player incorpora nuevas características lanzando actualizadas versiones al mercado.
Windows Media Player 12 contiene importantes cambios en la manera en que los archivos de medios se organizan y recuperan. Pero el cambio más importante se cree esta basado en el amigable menu que este presenta. Otra de las atracciónes es el aumento de velocidad en lascomunicaciónes con dispositivos como reproductores de MP3. Algunas de las características de sincronización automática se encuentran abilitadas para una coniderable cantidad de equipos.
