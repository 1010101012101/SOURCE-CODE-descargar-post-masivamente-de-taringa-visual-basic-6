Firefox, uno de los mejores navegadores Android, ya tiene una nueva versión Beta, la número 6. Se trata de una actualización más que interesante ya que trae muchas novedades: cambios estéticos, mayor velocidad de navegación, mejoras para tablets y en la calidad de las imagenes, etc.

Novedades Firefox Beta 6 para Android

* Nueva experiencia inicial: Firefox se inicia más rápido y usa menos memoria. Llega a tu página de inicio de forma rápida y descubre las características del navegador en los paneles laterales.
* Mayor calidad de imagen: Ahora podrás ver páginas Web e imágenes nítidas con menos pixelación
* Mejorada la experiencia Firefox en tablets de gran pantalla
* Mejoras en el asistente de formulario en Android
* Estilo fresco visual sobre Gingerbread OS
* Optimización para páginas que utilizan disposiciones rendering derecha a izquierda, por ejemplo, en hebreo y árabe
* Touch Events para una mejor interacción con las páginas Web
* IndexedDB permite a las páginas Web almacenar datos en línea para un acceso más rápido
* Separación automática de texto

Según comentarios de algunos usuarios, esta versión es algo más rápida que las anteriores.

Este navegador requiere Android 2.0 o superior.
