Requisitos del Sistema
* Windows® XP SP3 (32 bits), Windows Vista® SP1 o posterior (32 y 64 bits), Windows® 7 Home Premium, Professional o Ultimate (32 y 64 bits).
* Procesador de 2 GHz AMD o Intel®, 512 MB de RAM (1 GB de RAM para Windows Vista® o Windows® 7).
* Para la reproducción de discos Blu-ray y la edición de vídeo HD: procesador Intel® Core™ II Duo de 2,2 GHz, AMD Athlon 64 X2 5200+ 2,6 GHz o superior, con 2 GB de RAM.
* 5 GB de espacio en el disco duro para una instalación típica de todos los componentes (incluidas las plantillas, el contenido y el espacio en disco ocupado temporalmente).
* Se requiere tarjeta gráfica con aceleración 3D compatible con DirectX® 9.0, ajustes de color de 16 bits y un tamaño mínimo de memoria de vídeo de 128 MB (se recomienda 32 bits, color verdadero, 256 MB de RAM de vídeo dedicados y compatibilidad con Shader Model 3.0).
* Para la reproducción de discos Blu-ray: tarjeta gráfica NVIDIA 8600 y superior; ATI Radeon HD 3800 y superior; Intel G45 (Windows Vista/Windows 7); 512 MB de VRAM como mínimo; pantalla compatible con HDCP para la conexión digital.
* Unidad de DVD para la instalación y la reproducción; unidad compatible con discos Blu-ray para la reproducción de tales discos
* Unidad de CD, DVD o disco Blu-ray grabable o regrabable para la grabación y la creación de vídeos
* DirectX® 9.0c, revisión 30 (agosto de 2006).
* Reproductor de Windows Media® 9
* Los componentes de terceros (como Microsoft Windows® Installer 4.5, Microsoft .NET™ 3.5 SP1 y Microsoft Silverlight™ 3) se suministran con el producto o mediante descarga automática en el caso de que no se incluyan en el paquete.
* * Se precisa conexión a Internet para determinados servicios y para el uso de funciones BD-LIVE (se recomienda una conexión de ADSLo superior). Los costes derivados de la conexión a Internet son responsabilidad del usuario.

Nota:
