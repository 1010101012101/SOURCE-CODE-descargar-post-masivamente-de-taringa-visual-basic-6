Los Sistemas Windows XP SP3 VLK, Pro y Home x86 (32 Bits) tienen a su vez 3 metodos de instalacion distintos:


1- Instalacion completamente desatendida utilizando drivers SATA al estilo del 14_1
2- Instalacion semidesatendida utilizando drivers SATA al estilo Nlite con instalador XP-Total*
3- Instalacion personalizada utilizando un multipack de drivers SATA con instalador XP-Total* (permite recuperaciones en la consola de recuperacion y reparaciones por sustitucion completa de archivos pero no de configuracion).
4-El instalador XP-Total es un menu' exclusivo de este DVD en el que puedes elegir algunos componentes a instalar (similar a un WPI pero aparece mientras instalas Windows).
Los drivers SATA de los 3 tipos de instalacion son distintos.

Los sistemas Windows XP SP3 OEM (26 en 1) Pro y Home tienen a su vez 2 metodos de instalacion distintos:


1- Instalacion semidesantendida con drivers para SATA con un instalador para OEM similar al del XPOEM.rev1
2- Instalacion personalizada utilizando un multipack de drivers SATA con instalador XP-Total (permite recuperaciones en la consola de recuperacion y reparaciones por sustitucion completa de archivos pero no de configuracion).

El sistema Windows XP SP3 Student a su vez tiene 2 metodos de instalacion distintos:


1- Instalacion completamente desatendida utilizando drivers SATA al estilo del 14_1
2- Instalacion personalizada utilizando un multipack de drivers SATA con instalador XP

La instalacion de Windows XP SP3 x86 Media Center es semidesatendida utilizando drivers SATA con instalador XP-Total (incluye rollup2).

La Instalacion para Windows XP SP2 X64 es semidesatendida utilizando drivers SATA con instalador XP-Total (con opcion de realizar una instalacion completamente desatendida).

Los sistemas WinPE (Con acceso SATA Universal) y LiveCD se ejecutan desde el booteo y permiten por ejemplo acceder al sistema para recuperar datos cuando este no funciona.
