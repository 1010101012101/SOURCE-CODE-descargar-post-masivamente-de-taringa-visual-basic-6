Descripción
A principios de este año publique un videotutorial parecido, sin embargo Facebook deprecio el uso de Tag FBML, en este videotutorial se explica la implementacion de HTML, CSS y JavaScript para crear una aplicacion en Facebook y llamada via Iframe.

Se explica como agregar en la aplicacion la ventana de invitar amigos y como crear un menu tipo fb-tab.
Se recomienda ver el videotutorial publicado en enero del 2011 para darle mas utilidad a lo aprendido en este nuevo videotutorial.
