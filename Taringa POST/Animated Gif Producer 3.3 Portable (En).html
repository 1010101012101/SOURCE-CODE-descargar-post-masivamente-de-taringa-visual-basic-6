Ahora les traigo el Animated Gif Producer 3.3 Portable (En)... disfrútenlo...



Si usted está buscando un programa para diseñar animaciones GIF con alta calidad y fácil, Animated GIF Producer es una excelente opción para usted.

Animated GIF Producer es una herramienta rápida y muy fácil de usar para la creación de diversas animaciones, botones y banners que hará brillar su sitio web. Ya no es necesario gastar dinero en complejas plataformas de procesamiento de vídeo que le llevará un par de días para aprender a hacer una simple animación GIF. Con Animated GIF Producer, no hay complejidad y GIF se pueden crear en pocos minutos.



Creación de una animación GIF es una brisa. Lo puedes hacer mediante la mezcla de juntas imágenes, capturadas las instantáneas de vídeo y archivos de páginas múltiples. Para hacer que su animación se vea y se comportan de la manera que usted desea, puede personalizar una multitud de entornos, incluyendo la posición de fotogramas, el tamaño, y la demora entre muchos otros parámetros. También puedes aplicar efectos especiales a hacer una transición de una imagen a otra coherente y uniforme.


Animated GIF Producer incluye una interfaz intuitiva, donde todas las opciones pueden ser localizados fácilmente cuando los necesite. Usted puede agregar, procesar imágenes, organizarlas en una secuencia y configurar los parámetros de animación de salida deseada de un lugar con un punto y haga clic en la simplicidad. El programa le permite importar imágenes directamente desde el portapapeles. A esto se suma una instalación instantánea de captura incorporado para tomar instantáneas de videos. Una capacidad de exportación GIF a AVI también está disponible.

Características principales:

Importación de vídeo - incluyendo imágenes individuales;
Apoyo a AVI, DAT, MPG, MPE, MPEG formatos de video;
Un built-in instantánea facilidad de captura;
Soporte para imágenes BMP, JPEG, TIF, GIF, PNG, ICO, PCX, CUR, PSD de Adobe Photoshop y Windows formatos de imagen metarchivos;
Marcos de importación de otros GIF;
Cambiar el tamaño de los marcos (estiramiento);
Múltiples configuraciones de animación de salida;
Definir archivos GIF parámetros globales: tamaño de la pantalla, color de fondo;
Definir offset y retraso de forma global o por cada imagen por separado;
Adelante y hacia atrás para imágenes;
Aplicar efectos de transición para las imágenes;
Importación de imágenes desde el portapapeles;
Imágenes duplicadas;
Imágenes de reordenamiento;
Guardar las imágenes de archivo GIF-como un conjunto de formatos gráficos comunes;
Exportación de un archivo AVI;
Procesar las imágenes y ejecutar los efectos (de la imagen actual o todas las imágenes a la vez): cultivo, resample, girar y voltear imágenes en horizontal o vertical;

Editar una imagen seleccionada en un editor de imágenes externo.
