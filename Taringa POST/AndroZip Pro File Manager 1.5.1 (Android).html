AndroZip Pro File Manager es una excelente aplicacion para descomprimir archivos ZIP y otros archivos basados ZIP como APK, JAR, WAR, EAR, TAR, GZIP y RAR. AndroZip tambien puede crear archivos ZIP o basados en ZIP como APK, JAR, WAR, EAR, TAR y GZIP.

AndroZip Pro File Manager brinda funcionalidades de de gestor de archivos, permitiendo copiar, mover, borrar, renombrar archivos y carpetas, ver sus propiedades, asi como crear nuevas carpetas, enviar archivos adjuntos por correo electronico y abrir archivos entre muchas otras funcionalidades.

AndroZip File Manager funciona como un gestor y administrador de archivos, aplicacion de back-up, gestor de tareas, búsqueda de archivos, instalador de aplicaciones, enviador de archivos, instalador de aplicaciones.

Caracteristicas AndroZip Pro File Manager
•Administrar archivos/carpetas: Selección simple y múltiple, copiar, borrar, mover, ordenar
•ZIP, RAR, 7ZIP, GZIP, TAR, BZIP2
•Instalar aplicaciones
•Menús de tap corto y tap largo
•Administrar aplicaciones: hacer copias de seguridad, desinstalar
•Administrador de Tareas: Ver aplicaciones activas

Novedades AndroZip Pro File Manager 1.5.1
•Mejoras en el rendimiento de la aplicación
•Corrección de varios errores

Esta aplicacion requiere Android 1.6 o superior.
