Microsoft Office 2010 Professional Plus (nombre código Office 14) es una suite ofimática creada por
Microsoft para Microsoft Windows, sucesora de Microsoft Office 2007. En esta nueva versión
se ha actualizado la interfaz del usuario e incluye nuevas funcionalidades a la vez que mejoras de
compatibilidad con nuevos formatos de archivo.




Requisitos mínimos
- Procesador: Procesador de 500 MHz o superior
- Memoria: 256 MB de RAM o superior
- Disco Duro: 1.5 GB, parte del espacio se liberará después de terminada la instalación.
- Video: Tarjeta grafica con 64 MB de memoria o superior (aceleración de hardware gráfico requiere DirectX 9.0c)
- Sistema operativo: Windows XP con Service Pack 3, Windows Vista con SP1, Windows 2003 R2, Windows Server 2008 con SP2 y Windows 7
