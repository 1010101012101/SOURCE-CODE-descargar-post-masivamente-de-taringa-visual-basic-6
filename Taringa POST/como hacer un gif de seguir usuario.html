descipcion del programa

CaptureWizPro es una potente herramienta de captura de pantalla capaz de copiar prácticamente cualquier cosa que aparezca en tu monitor, tan fácil como hacer una foto.

El propio programa te permite comprimir las imágenes para enviarlas por e-mail, reunir diversas capturas en el interfaz con imágenes reducidas (thumbnails), imprimirlas y guardarlas en formato BMP, JPG, PNG y GIF.

Con esta herramienta serás capaz de hacer capturas de cualquier aplicación windows, documentos, mensajes de error, páginas web e incluso fotogramas de vídeos en formato de Media Player y Real Player.

El programa es muy fácil de usar: sólo tienes que mover la ventana de captura sobre el texto o imagen que quieres copiar, ajustar el tamaño y con un simple clic capturarla, imprimirla, enviarla por e-mail, editarla o pegarla en el Portapapeles.

Sistemas operativos
Windows 2000, Windows 2003, Windows 95, Windows 98, Windows ME, Windows NT, Windows Server 2008, Windows Vista, Windows XP

Peso del archivo: 5.5 mb ya cuando ah sido descomprimido.
