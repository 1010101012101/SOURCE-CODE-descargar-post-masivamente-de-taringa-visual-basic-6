Virtual PC 2007 SP1


Microsoft ha publicado una nueva versión de su software de virtualización gratuito incluye ahora soporte para Windows Vista SP1 y Windows XP SP3 tanto como sistemas cliente como host. Esto significa que puedes ejecutar el entorno de virtualización con estos sistemas operativos y emularlos igualmente. Como su predecesor, Virtual PC 2007 SP1 no soporta oficialmente las versines Home, Home Premium o Home Basic de Windows XP Home Edition de Windows Vista. Eso si, la mayoría de ellas pueden ejecutarse bastante bien a pesar de ello. Lo único es que verás un mensaje de aviso que te dice que estás usando un sistema operativo no soportado - las incongruencias que tiene Microsoft algunas veces.
