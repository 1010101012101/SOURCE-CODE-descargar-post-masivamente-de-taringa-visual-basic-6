Tu Computadora más segura que nunca




WinLock : es un programa integral de seguridad para ordenadores personales o de acceso público compatible con todas las versiones de Windows desde 95 hasta Seven.

Es un programa que asegura que sólo las personas autorizadas pueden acceder a información sensible sobre su computadora. Esta herramienta de seguridad le permite restringir a sus usuarios el acceso a los recursos importantes del sistema como el Panel de control, escritorio y registro del sistema, desactivar las teclas de acceso rápido de Windows (por ejemplo, Alt- Ctrl-Del, Alt-Tab, Ctrl-Esc, etc).


Con WinLock usted puede controlar cuánto tiempo pueden utilizar su ordenador. Se ejecuta desde la bandeja del sistema y requiere una contraseña para acceder a las opciones disponibles. Se carga automáticamente con Windows y le permite agregar un mensaje de arranque opcional, una notificación de audio, y establecer el límite de tiempo.

Una vez alcanzado ese límite, Windows es sumariamente cerrado. Puede cambiar el temporizador de encendido y apagado.

También le permite denegar el acceso a archivos seleccionados, aplicaciones y ventanas del Explorador (la papelera de reciclaje, Mi PC, etc), USB y discos duros, filtrar el contenido de Internet y prohibir el acceso a sitios web cuestionables.

Cuando se activa, no hay manera de evitarlo sin la contraseña correcta. Si usted está preocupado acerca de sus hijos arruinando sus archivos de trabajo, WinLock es una solución de software exclusivamente para sus necesidades de seguridad.
