¿Como Activarlo?
El windows lleva su activador, muy fácil de usar, solo elegir la versión que tienen de windows 7 (español o ingles) y también a cuantos bits corre si 32(x86) o 64(x64) bits y seguir las instrucciones, sirve para cualquier versión de windows 7.
¿Cómo traducirlo al Español?
Luego de aplicar el activador, acceder a Windows Update desde el menú inicio. Buscan actualizaciones y en ‘actualizaciones opcionales’ encontrarán el MUI en español para descargar.
Una vez descargado el MUI, vayan a: Control Panel, Clock Language and Region, Change display Language, Install/Unistall language, abajo de esa opcion esta Ingles y Español, eligen Español, luego yes a lo que les sale, regresan a la pagina de Bienvenida de Windows entran de nuevo y listo.
