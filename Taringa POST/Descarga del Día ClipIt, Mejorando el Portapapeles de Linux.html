Descarga del Día ClipIt, Mejorando el Portapapeles de Linux




El portapapeles es una de esas características de todo sistema operativo que, aunque no le prestamos atención, son necesarias y hacemos un intensivo uso de ellas cada vez que estamos delante de un ordenador. Aunque parezca increíble, poco hemos avanzado en el uso de este añadido y se le puede sacar mucho partido.

ClipIt es un programa para Linux que mejora el portapapeles por defecto de éste sistema y añade funcionalidades tan interesantes como acumular varios elementos en un historial (no sólo la última), poder buscar entre las últimas cosas copiadas o ejecutar acciones con las cosas que copiamos.

En la última versión, recién actualizada, se han incorporado novedades:

* El teclado numérico ahora sirve para seleccionar elementos del historial.
* Mejoras en el icono de la barra de tareas.
* Permite crear elementos estáticos, que siempre se encuentren disponibles en el portapapeles (códigos, URLs, IP…)
* Menú embebido en el icono de la barra de tareas

Y por supuesto los consiguientes arreglos de fallos que tenía la anterior versión.




Para instalarlo en Ubuntu lo tenemos muy sencillo ya que existe un repositorio que podemos añadir a nuestra instalación:

sudo add-apt-repository ppa:shantzu/clipit
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 521767F1
sudo apt-get update

Y lo tendremos formando parte de nuestro sistema.
