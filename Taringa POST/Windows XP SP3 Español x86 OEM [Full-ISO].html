2- El ABC de Windows

2.1- Versiones

Existen 3 versiones de Windows, que son comercializadas según el segmento al que fue desarrollado.

- Oem: Para empresas (un solo serial para varias pcs)

- Rtm: Para fabricantes (un solo serial para una pc)

- Retail: Para usuarios finales (un solo serial para una pc)

Hay que entender que mientras las OEM trabajan con un solo serial que ésta registrada para una empresa. la RTM son las que vienen cuando compras tu equipo en tienda, y finalmente las Retails que son las que adquieres en tienda especializadas en softwares.
