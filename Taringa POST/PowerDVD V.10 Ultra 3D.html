Requisitos del Sistema

Los requisitos del sistema pueden depender de la película que está mirando.
Las nuevas tecnologías 3D necesitan hardware adicional, para ver películas en 3D.
Vea la página Hardware para más información.

CyberLink PowerDVD optimiza el rendimiento al seleccionar el algoritmo de
desentrelazado de vídeo apropiado para la potencia de procesamiento disponible.
CyberLink PowerDVD mantiene su compatibilidad para procesadores inferiores a 1,6
GMz al garantizar que la Tecnología TrueTheater™ se desactivará por defecto.
Las tecnologías de vídeo y audio avanzadas de CyberLink PowerDVD funcionan mejor
cuando se utilizan con procesadores superiores a 2,4 GHz.

Requisitos del sistema para discos Blu-ray, DVD y reproducción TrueTheater™ 3D


Sistema Operativo

* Microsoft Windows 7, Vista* y XP SP3* (se requiere Windows XP Service Pack 3)

Procesador (CPU)

* Reproducción de Disco Blu-ray: Pentium 4 541 (3.2 GHz), Athlon 64 X2 3800+ o superior
* Reproducción TrueTheater™ 3D: Pentium D 950 (3.0GHz), AMD Athlon 64 X2 3800+ o superior
* Reproducción DVD: Pentium 4 (2.4 GHz), Athlon 64 2800+ (1.8 GHz) o superior

Procesador Gráfico (GPU)

* Reproducción Disco Blu-ray: Intel G45, ATI Radeon HD 2400, NVIDIA Geforce 7600GT o superior
* Reproducción TrueTheater™ 3D: Intel G45, ATI Radeon HD 2400, NVIDIA Geforce 8800GT o superior
* Reproducción DVD: accelerador gráfico AGP o PCI Express soportando DirectX 9.0 o superior

Nota: Le recomendamos actualizar el controlador de su tarjeta gráfica a la última versión. Una pantalla 3D es necesaria para reproducción TrueTheater™ 3D.

Memoria del sistema (RAM)

* Reproducción Disco Blu-ray: 2 GB necesarios para Vista y Windows 7; 1 GB necesario para Windows XP
* Reproducción TrueTheater™ 3D y DVD: 1 GB necesario para Vista y Windows 7; 512 MB necesarios para Windows XP

Espacio Disco Duro

* 300 MB para instalación de producto

Dispositivo de visualización

* Reproducción Disco Blu-ray y DVD: pantalla compatible HDCP para visualización digital. TV o pantalla de computadora para visualización analógica.
* Reproducción TrueTheater™ 3D: pantalla y gafas 3D necesarias. Pantalla compatible HDCP para visualización digital. TV o pantalla de computadora para visualización analógica.

Unidad de disco

* Reproducción Disco Blu-ray: reproductor de disco combo BD-ROM/BD-RE o Super Multi Blu
* Reproducción TrueTheater™ 3D y DVD: reproductor de disco combo DVD-ROM, DVD+-R/RW, DVD-RAM o DVD Super Multi

Micrófono

* Un micrófono es necesario para grabación de voice-overs en el modo Movie Remix.

Connexion Internet

* Una conexión a Internet es necesaria para acceder a los servicios web en línea.
