Esta aplicación es un tema Android para MXHome Launcher, por lo cual para poder aplicarlo deberás tener instalado MXHome Launcher.

MXHome Theme BlueWatch es un tema con estilo reloj sensual y moderno.

Características MXHome Theme BlueWatch

•Esta aplicación proporciona un efecto 3D de gota de agua y animación G-sensor.
•El usuario sentirá la sensación dinámica y mecánica de partículas en 3D y animación G-sensor.
•Pantalla personalizada dará servicio más realista para los usuarios finales.
•El establecimiento de iconos de acceso directo del índice de tiempo brindará una experiencia única.

Este tema no posee soporte HVGA (320×480).

Este tema requiere Android 2.1 Eclair o superior.
