Requisitos: Android 2.1 o superior

El uso de VPN tanto? Pruebe una VPN, puede conectarse en un clic y mantenerse conectado hasta que usted decida.
Conexión a VPN nunca ha sido tan fácil.

Muchas personas utilizan VPN para acceso remoto seguro, escritorio remoto, o simplemente para los sitios web de desbloqueo. El soporte nativo para Android VPN requiere un montón de tapping, y escribir una contraseña para conectarse.
Con una VPN puede establecer una conexión VPN en un solo toque. Además, el plug-in Tasker / locale función le permite establecer automáticamente las conexiones VPN.

Características:
+ Múltiples cuentas VPN.
+ Guardar nombre de usuario / contraseñas
+ VPN estado widget
+ 1 toque en flash para conectar / desconectar
+ Reconexión automática, cuando se pierde la conexión
+ "Mantener viva pings" para mantener viva la conexión
+ Conectar automáticamente, utilizando Tasker / locale plug-in
+ Importación / exportación de sus perfiles de una VPN. Útil para el despliegue a escala grande de copia de seguridad o una VPN.
+ Compatible con: PPTP, L2TP, L2TP/IPSec CRT
- L2TP o IPSec secretos PSK (Pre Shared Key) no son compatibles. Usted puede utilizar esta aplicación para guardar la contraseña, pero usted tiene que conectarse a través de la aplicación para Android VPN.
