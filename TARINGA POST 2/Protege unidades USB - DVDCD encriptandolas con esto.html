Protege unidades USB - DVD/CD encriptandolas con esto

Master Voyager fue especialmente diseñada para crear una contraseña protegida de DVD / CD y discos USB Flash Drives (Pendrives).

Protegidos los medios de comunicación es totalmente autónomo y no necesita ningún software especial instalado en la computadora que se opera.

Puede crear discos protegidos con almacenamiento en el equipo de instalación y "reproducir" o abrir archivos en cualquier otro ordenador (requiere Windows 7, Windows Vista o Windows XP).

Master Voyager aplicaciones y usos:

•Protegido por contraseña CD / DVD con grandes cantidades de datos sensibles puede ser con seguridad enviados por correo postal habitual.

•Copias de seguridad cifradas en CD / DVD se puede mantener en lugares públicos e inseguridad.

•En el caso de robo de la pérdida, cifrada unidades flash USB y la información confidencial en ellos se mantendrá confidencial.

•Basta con introducir la contraseña y trabajar con cualquier documento sobre la unidad de DVD / USB Flash como de costumbre.
Main Features: Características principales:

•No es necesario para descifrar los datos de DVD / CD / USB o copiar archivos desde CD / USB al disco duro. Basta con introducir una contraseña y trabajar con encriptado USB o CD

Abrir documentos cifrados directamente desde el CD / USB e incluso poner en marcha programas o reproducir películas.

Master Voyager utiliza el exclusivo sobre la marcha-la tecnología de descodificación desarrollado exclusivamente para CryptoExpert Professional .

•Confidencialidad de datos ultra segura.

Master Voyager utiliza el cifrado fuerte de todos los datos que utiliza AES de 256-bit de protección.

AES es el Federal Information Processing Standard (FIPS) que se utiliza para proteger la información más sensible del gobierno estadounidense y organizaciones comerciales, incluidos los bancos.

•Integrados de CD / DVD módulo quema para crear DVD o CD encriptados .

Master Voyager apoya CDR / CDRW / DVDR / DVDRW tipos de disco.

Sólo tiene que seleccionar los archivos / carpetas que necesita para almacenar de forma segura en cualquier medio de almacenamiento principal y dejar Master Voyager hacen el resto.

•No hay necesidad de receptores disco para funcionar Master Voyager.

Si envía un disco o una memoria USB a través del correo electrónico a un tercero, que no es necesario para operar Master Voyager, simplemente se necesita la contraseña para desbloquear los datos protegidos y la información.

El tercer usuario simplemente tiene que insertar el medio de almacenamiento a su ordenador e introduzca la contraseña para trabajar como lo harían con cualquier otro CD o dispositivo USB.

El único requisito es tener instalado Windows 7/Vista/XP (de 32 bits y 64 bits son compatibles).

No hay ninguna descarga de software necesarias y no hay pérdida de tiempo reiniciando la pc o por configuraciones de equipos de que preocuparse, el proceso es sencillo y automatizado.

•Master Voyager 2.00 y anteriores proporcionan la capacidad de trabajar con los medios de comunicación Encrypted USB disco / DVD sin derechos de administrador

Autoplay módulo de partición Explorer permite a los usuarios trabajar con cifrado unidad flash USB / DVD como si fuera un archivo encriptado.

•Master Voyager 2.20 y anteriores se prevé la creación de seguro de particiones de mas de 4 GB en Unidades Flash USB formateado en NTFS o exFAT
