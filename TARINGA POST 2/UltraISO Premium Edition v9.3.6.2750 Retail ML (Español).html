Descripción
UltraISO es una herramienta para crear, editar y convertir archivos de imágenes de CD a formato ISO estándar y otros.
Usando UltraISO puedes abrir directamente los archivos de imagen ISO, extraer el documento deseado, editarlo y transformar ese archivo de imagen al tipo estándar ISO, también permite crear archivos ISO desde CD-ROM o discos duros. Al mismo tiempo puedes procesar la información ISO para la carga inicial del ordenador, y crear así CDs para esta función de arranque.
