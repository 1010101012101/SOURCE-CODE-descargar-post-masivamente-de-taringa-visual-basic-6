DAEMON Tools es un emulador de imágenes de disco propietario para el sistema operativo Microsoft Windows. La aplicación permite al usuario montar imágenes de CD y DVD como unidades virtuales. También incluye varios mecanismos para intentar evitar algunas de las protecciones anticopia más extendidas. 

DAEMON Tools originalmente fue una mejora de otro software llamado Generic Safedisc emulator, del cual ha heredado todas sus características y funcionalidades. 

Formatos de imagen de disco soportados 
* bin/cue - CDRWin 
* b5t - BlindWrite 5 
* bwt - BlindWrite 
* ccd - CloneCD 
* cdi - DiscJuggler 
* iso - Imagen de CD estándar 
* mds - Alcohol 120% (Media descriptor files) 
* nrg - Nero 
* pdi - Instant CD/DVD
