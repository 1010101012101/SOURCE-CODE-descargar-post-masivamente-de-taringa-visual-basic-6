La Forma mas sencilla de crear tu propia pagina Web!

EZGenerator proporciona al usuario inexperto la forma más simple de crear sitios web sin necesidad de conocimientos sobre HTML y programación.

El programa incluye más de 3000 plantillas para todo tipo de páginas web que podrás respetar o personalizar para adaptarla a tu propio diseño.
Blogs, fotoblogs, calendarios, podcasts, comercio electrónico, encuestas flash, presentaciones de imágenes... Éstas y otras aplicaciones web podrás construir con EZGenerator.

Lo único que tienes que hacer es elegir la plantilla y comenzar a decidir aspectos como el tipo de tipografía o las imágenes que quieres reemplazar por las tuyas propias. De esta forma, en poco tiempo podrás disfrutar de tu propia página web.
EZGenerator incluye su propio cliente FTP, por lo que no tendrás que recurrir a otro programa para subir los archivos al sitio.
