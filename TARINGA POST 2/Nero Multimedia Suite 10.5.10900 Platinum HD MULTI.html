Requerimientos:

- Windows XP SP3 (32 bits), Windows Vista SP1 o posterior (32 y 64 bits), Windows 7 Home Premium, Professional o Ultimate (32 y 64 bits).
- Procesador de 2 GHz AMD o Intel, 512 MB de RAM (1 GB de RAM para Windows Vista o Windows 7).
- Para la reproducción de discos Blu-ray y la edición de vídeo HD: procesador Intel Core II Duo de 2,2 GHz, AMD Athlon 64 X2 5200+ 2,6 GHz o superior, con 2 GB de RAM.
- 5 GB de espacio en el disco duro para una instalación típica de todos los componentes (incluidas las plantillas, el contenido y el espacio en disco ocupado temporalmente).
- Se requiere tarjeta gráfica con aceleración 3D compatible con DirectX 9.0, ajustes de color de 16 bits y un tamaño mínimo de memoria de vídeo de 128 MB (se recomienda 32 bits, color verdadero, 256 MB de RAM de vídeo dedicados y compatibilidad con Shader Model 3.0).
