Rainlendar 2.9 Beta 107

Rainlendar es un calendario personalizable que muestra el mes actual. Es una aplicación muy ligera que no utiliza muchos recursos del sistema o toma mucho espacio en su escritorio.

Pequeño y ligero
Diferentes eventos de tipo puede tener apariencia diferente
Compatible con Windows transparencia
Sincroniza los eventos entre varios clientes
En múltiples idiomas
Muestra una alarma cuando un evento se debe
Soporta archivos de iCal.
Muestra las citas de Outlook
Incluye una lista de todo-
Funciona también como plugin Litestep
Muestra la fecha actual en el icono de la bandeja
Se pueden pegar en el escritorio (es decir, no se esconde con Mostrar escritorio)
Teclas para acceso rápido
Pelado fácil con una interfaz de usuario

Cambios Recientes:

- Mac: Importar archivos con letras extranjeras en el nombre del archivo debería funcionar ahora.
- Las ventanas ocultas no se ha establecido ya para mejorar el rendimiento.
- archivos sin conexión no funciona con el calendario de Google a menos que la lista de tareas fue leído desde el servidor también.
- Si el icono de la bandeja no está seleccionada para la piel se utiliza el icono predeterminado. El icono se puede desactivar en las opciones avanzadas.
- El cursor del ratón correctamente a la mano cuando está sobre un enlace clikable.
- El archivo skin.xml se intenta leer de la primera subcarpeta dentro del archivo zip.
- eventos de Google que se marcados como no modificables se establecen como de sólo lectura en Rainlendar.
- Los calendarios de la red no se leen inmediatamente más si la duración de la encuesta ha pasado, mientras que el equipo fue suspendido.
- La hora de vencimiento no se almacenan adecuadamente para recordar la lista de la leche de tareas.
- La siguiente alarma para las tareas que se repitió días la última semana del mes se calculó de forma incorrecta.
- Cambio de la fecha de vencimiento o el tiempo se restablecerá el tiempo de completar la tarea.
- La información sobre herramientas no se muestran más de los elementos que se establecen ocultos.
- Se ha añadido una opción para seleccionar el archivo de configuración de copias de seguridad si la puesta en marcha ha fallado.
- Se ha añadido una opción avanzada para eliminar el debido tiempo de la ventana de alarma. Esto también se desactiva la ventana de avisos de actualización.
- Añadido soporte para las tareas de sub. Las tareas se pueden mover en otra tarea en la lista tiene orden personalizado.
- La posición de la tarea de realizar el pedido de ahora se escribe con la propia tarea.
