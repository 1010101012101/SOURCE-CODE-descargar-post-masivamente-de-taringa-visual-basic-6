Celestial Bodies son Live Wallpapers 3D del sistema solar, los cuales son altamente personalizables.

Como abrir

Inicio> Menú> Fondos> Wallpapers> Live Wallpapers

Al igual que con todos los live wallpapers, este no se abrirá desde la pantalla del Market. Esto no quiere decir que no está bien instalado! Simplemente debes ir a la configuración de wallpapers como se describió aquí arriba y selecciona Celestial Bodies. Desde allí se puede ajustar la configuración y aplicar el wallpaper. Lea las instrucciones que se incluyen (en la página de configuración) para una explicación completa de las funciones disponibles.

Caracteristicas Celestial Bodies LiveWallpaper

• Movilidad total de los planetas, así como el complejo movimiento del punto de vista proporcionan un fondo muy dinámico para tu dispositivo
•Posibilidad de elegir entre nuestro sistema solar o un sistema generado de forma aleatoria con numerosas configuraciones variables para una amplia variedad de diferentes escenarios posibles cada vez que la pantalla se despierta
•Varios tipos de interacción táctil
•Configuración fácilemente comprensible que permite control total del usuario en la personalización de varios aspectos del sistema solar, vista e interacción
•Ajustes de procesamiento modificables para un mayor rendimiento o una mayor autonomía
•Soporte de visualización de música
