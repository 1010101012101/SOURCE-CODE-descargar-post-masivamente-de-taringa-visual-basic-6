Diagnostico de redes facilmente con este programa

Si habitualmente experimentan retrasos o cuellos de botella cuando se navega por Internet o la intranet, sabes lo frustrante que puede ser.

NetInfo analiza tu conexión y diversos aspectos de la Internet con el fin de aislar posibles problemas.

Una vez que se determina dónde está el problema, NetInfo te ofrece una gran cantidad de herramientas que le ayudan a resolverlo.

Si el problema está más allá de su ámbito de control y no se sabe a quién contactar en ello, NetInfo le dará un nombre, una dirección de correo electrónico y número de teléfono.
















Con herramientas como Información Local, Conexiones, Ping, Trace, de búsqueda, Finger, Whois, tiempo durante el día, cita, HTML, escáner y Servicios, E-mail y web del Centro, se le pondrá en contacto con su interior geek NetAdmin.

Información Local - Muestra las direcciones del sistema IP y direcciones MAC.
Muestra todos los locales de conexiones TCP / IP y su dirección en el extranjero de conexión IP.
Ping - Le permite comprobar rápidamente la conectividad de red a otra dirección IP en la red.
Trace - Muestra una lista de routers entre el sistema y el sistema de destino.
Lookup - Le permite realizar DNS de avance y retroceso consultas de búsqueda.
Finger - Muestra información sobre los usuarios actualmente conectados a un servidor determinado.
Whois - le permite consultar información de contacto para los dominios registrados con un servidor de Whois.
Daytime - muestra la hora local del día para cualquier host remoto especificado.
Time - le permite consultar el valor de tiempo de un servidor de tiempo remoto y sincronizar el valor de tiempo con el reloj del sistema local.
Quote - Le permite ver las citas de un servidor remoto Cita.
HTML - Muestra el código HTML de cabecera y los datos de la página para cualquier dirección de la página Web especificada.
Scanner - Escanea en busca de nombres de host para un determinado rango de direcciones IP y muestra el estado de cada dirección IP de la gama.
Services - Lista de los servicios de red disponibles (HTTP, Telnet, DNS, SMTP, etc) para un host en particular.
E-mail - Le permite verificar cualquier dirección de correo electrónico con sólo introducir un nombre principal de usuario completo (user@domain.com).
Web Center - Muestra las herramientas disponibles en línea, como tos, consejos y noticias.
