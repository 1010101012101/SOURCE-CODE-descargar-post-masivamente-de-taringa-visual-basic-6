Comodo Programs Manager es la propuesta de Comodo para ver, reparar y desinstalar las aplicaciones que tengas instaladas en Windows, así como controladores, servicios del sistema y actualizaciones.

Comodo Programs Manager es muy fácil de usar. Muestra las diferentes listas, con buscador integrado. Desde ahí podrás obtener información de cada programa o servicio y decidir si desintalarlo, repararlo o realizar cambios, tal como hace la opción Agregar y quitar programas de Windows.

Comodo Programs Manager muestra también las actualizaciones de Windows instaladas y las disponibles.

Por lo demás, esta primera versión de Comodo Programs Manager no ofrece nada nuevo respecto al gestor por defecto de Windows. La opción de backup no funciona bien, así como el apartado Windows Features, que se muestra en blanco.
