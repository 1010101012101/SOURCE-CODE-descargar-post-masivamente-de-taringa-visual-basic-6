Notepad++
Es un editor de códigos y block de notas que soporta varios lenguajes de programación, para el entorno de MS Windows.
Basado en el poderoso Scintilla, Notepad++ está escrito en C++ y usa solamente Win32 API y STl lo que asegura una mayor velocidad de ejecución y un menor tamaño del programa. Optimizando la mayor cantidad posible de rutinas sin perder amigabilidad, NOtepad++ esta tratando de reducir la emisión de dióxido de cárbono del mundo. Mientras usas menos poder del CPU, la computadora puede ahorrar energía o consumir menos, resultando un mejor ambiente. Soporta JAVA, C, C++, C#, BASIC, etc.
