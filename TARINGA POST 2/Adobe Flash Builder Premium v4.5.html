¿Qué es Flash Builder 4.5?
Adobe® Flash® Builder® 4.5, anteriormente Adobe Flex® Builder™, es una herramienta de desarrollo basada en Eclipse™ que permite crear en muy poco tiempo aplicaciones móviles, web y de escritorio de gran expresividad haciendo uso de ActionScript® y del marco de trabajo de código abierto Flex.

Experiencias ricas y expresivas
Cree aplicaciones más intuitivas y atractivas para ayudar a los usuarios a comprender y aprovechar los datos y contribuir, de esta manera, a reforzar las principales actividades de negocio. Esto redundará en un aumento de la productividad y la efectividad.Aplicaciones móviles para Android, iOS y RIM
Saque provecho de la nueva compatibilidad con el desarrollo y las pruebas de aplicaciones para dispositivos móviles, que hace posible el uso de un código base común para crear aplicaciones para Android™, Apple iOS y Blackberry Tablet OS, al tiempo que se comparte el código de las aplicaciones web.
Agrupación y firma de aplicaciones móviles
Implante, agrupe y firme los recursos necesarios como archivos de instalación para plataformas específicas y podrá cargarlos en un sitio o tienda de distribución de aplicaciones para dispositivos móviles.
Generación de código
Utilice la asistencia/corrección rápidas para cambiar el nombre y manipular variables, generar captadores/establecedores y controladores de eventos, organizar importaciones, y mucho más. Utilice la función de anulación/implementación para definir los métodos desde una superclase o interfaz.
Resaltado en directo de referencias no válidas
Consulte las referencias a variables, métodos, clases e interfaces no válidas a medida que escribe en el editor de código y, a continuación, utilice las funciones de asistencia/corrección rápidas para generar código auxiliar.
Plantillas de código
Utilice más de 100 nuevas plantillas de código de prácticas recomendadas (también conocidos como fragmentos) para ActionScript, MXML y CSS. Revise e inserte plantillas utilizando la asistencia de código. Cree plantillas personalizadas, e impórtelas y expórtelas para compartirlas entre los equipos.
Flujo de trabajo bidireccional entre diseñador y desarrollador
Aproveche la nueva compatibilidad de Flash Builder 4.5 y Flash Catalyst CS5.5, la cual permite a los diseñadores abrir, realizar cambios y guardar proyectos de Flex que, o bien se crearon o se editaron previamente en Flash Builder.
