Buenas a todos, les dejo un excelente programa muy muy liviano (200Kb) y portable, para optimizar la conexion de internet que tengas.

El optimizador de TCP (TCP Optimizer) es un programa diseñado para proporcionar una interfaz sencilla e intuitiva para el ajuste parámetros relacionados de TCP e IP en el Registro de Windows. Tiene en cuenta todos los RFCs relacionados con el Microsoft TCP / IP rarezas de aplicación, verifica todos los lugares de registro para los mismos parámetros TCP / IP (compartido ICS, AOL protocolo MTU, etc), Y modifica los parametros necesarios en el registro. Con la configuración Optimizada se logrará excelentes resultados en la velocidad de conexion.
