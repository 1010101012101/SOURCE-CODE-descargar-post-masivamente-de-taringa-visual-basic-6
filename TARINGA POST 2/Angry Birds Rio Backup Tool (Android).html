¿Eres fanático de Angry Birds? Pues entonces esta aplicacion es ideal para ti. Angry Birds Rio Backup Tool es una interesante herramienta que te permitirá hacer copias de seguridad y restaurar los juegos guardados y las mejores puntuaciones de Angry Birds Rio a la tarjeta SD.

NOTA: para realizar el backup o restaurar, deberás completar al menos un nivel. No puedes simplemente instalar el juego y restaurar.

Esta aplicacion requiere Android 1.6 o superior.
