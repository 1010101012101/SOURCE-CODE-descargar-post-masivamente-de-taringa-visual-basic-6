Descripción:

Acronis True Image Home 2011, permite asegurar los datos importantes, incluidas las imágenes, música, documentos y aplicaciones, y pueden recuperarse fácilmente en caso de catástrofe. Mediante una interfaz gráfica intuitiva, los usuarios pueden definir fácilmente con qué frecuencia quieren hacer copias de seguridad de un PC.

Acronis True Image Home 2010 ofrece a los usuarios domésticos un sistema de copia de seguridad y recuperación de su sistema operativo, aplicaciones, configuraciones y archivos personales que es fiable y rápido. Además, Acronis Nonstop Backup™ ofrece protección continua de sistemas y datos, que impide la pérdida de éstos.

Caracteristicas y Tecnologias:

*Creación de imágenes de disco.
*Copia de seguridad de archivos.
*Copias de seguridad incrementales y diferenciales.
*Copia de seguridad de archivos: Realice copias de seguridad de archivos, carpetas o categorías de archivos aislados.
*Copias de seguridad activas: siga trabajando mientras se realiza la copia de seguridad.
*Try y Decide: Crea un entorno seguro donde podràs probar productos nuevos sin poner en riesgo tu PC.
*Cifrado de imágenes con AES.
*Arranque y restaure un sistema sin utilizar un medio de arranque adicional.
*Protección con un solo click.
*Excluye archivos y carpetas: Respalda sòlo los datos que quieras.
*Copia de seguridad de reserva.
*Configure una vez y realice copias de seguridad automáticamente.
*Acronis Secure Zone: Proteja su sistema guardando una imagen en una partición oculta especial del disco duro.
*Acronis Startup Recovery Manager: Arranque y restaure un sistema después de un fallo.
*Copias de seguridad Set y Forget: Configure las copias de seguridad una sola vez y realícelas de manera automática.


Nuevas Funcionalidades:

*Integración completa con Windows 7: Descubra cómo Acronis True Image Home 2011 y Windows® 7 funcionan conjuntamente para proteger mejor su PC.
*Compatibilidad con USB 3.0: Acronis True Image Home es compatible con el nuevo estándar USB 3.0.
*Esquemas de copia de seguridad predefinidos: Acronis True Image Home le ofrece esquemas de copias de seguridad predefinidos que se han diseñado para cubrir necesidades concretas de los usuarios.
*Puntos de comprobación para Acronis Backup Explorer: Solo lo encontrará en Acronis: una manera fácil de ver y gestionar los archivos de copia de seguridad desde una única pantalla.
*Acronis Nonstop Backup: Crea copias de seguridad incrementales cada cinco minutos automáticamente, lo que permite a los usuarios recuperar versiones de sus archivos y carpeta en cualquier momento del pasado.
*Compatibilidad con discos duros virtuales: Convierta las imágenes de copia de seguridad de Acronis (.tib) en archivos de disco duro virtual (.vhd) y viceversa, lo que permite a los usuarios ejecutar los sistemas existentes como máquinas virtuales.
*Programador nuevo y eficaz: El nuevo programador amplía el número de opciones de programación disponibles. Use la nueva vista de calendario para crear programaciones y tareas. Suspenda y active tareas, indique las fechas en que deben ejecutarse las tareas y copie las tareas en otras fechas.
*Arranque desde una imagen de Windows 7: Los usuarios de Windows 7 Ultimate pueden arrancar desde una imagen de copia de seguridad (archivo .tib) de una partición de su sistema. Esto permitirá poner a prueba la capacidad de arrancar un sistema de copia de seguridad sin realizar una restauración real.
