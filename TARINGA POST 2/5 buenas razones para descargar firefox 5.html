Mozilla ofrece a los usuarios más control sobre sus datos de navegación en la Web al permitir que por medio del navegador con la función No seguir opten por el seguimiento online. Además, en Firefox 5 es aún más fácil de encontrar la sección de preferencias.

También hay que resaltar que Firefox 5 para Android incluye la característica No seguir, por lo que Firefox 5 es el primer navegador que soporta esta función en múltiples plataformas, afirmó Mozilla.

2. Es pequeño y rápido

Firefox 5 fue más rápido que Chrome y Opera en la prueba de SunSpider JavaScript, (no se incluyeron otros navegadores), y su tiempo de arranque en frío es comparable con la actual versión de Google Chrome. Firefox 5 también destacó por su buen manejo de la memoria RAM, pudimos abrir 25 sitios web sin problemas.

3. Es compatible con una red abierta
Con el apoyo de estándares mejorados para HTML5, XHR, MathML, y SMIL , Firefox 5 es la mejor opción para un navegador Web abierto. El nuevo Firefox Add-on SDK para Windows, Mac y Linux permite el desarrollo local de complementos. Por su parte, el Firefox Add-on Builder Beta ofrece una interfaz basada en Web. La nueva versión del software multiplataforma también incluye soporte para animaciones CSS.

4. Contiene muchas mejoras

Son muchas las mejoras las que se han incluido en Firefox 5. No todas pueden ser notables en la interfaz de usuario, pero sin duda se van notando conforme el usuario usa el navegador.

La versión de Firefox 5 para Android por su parte incluye una serie de correcciones de errores, mejoró la velocidad de carga de la página - especialmente en las redes 3G - y añade soporte para IPv6.

5. Es segura, independiente y orientado al usuario

Firefox es un software de código abierto, lo que significa que se desarrolla con la ayuda de los usuarios, por lo que los problemas de seguridad tendrán un seguimiento continuo que lo realizará una comunidad global de usuarios y desarrolladores.

Firefox claramente enfrenta a un fuerte Chrome de Google, que tal vez tenga verdaderos problemas con el próximo Firefox 6.
