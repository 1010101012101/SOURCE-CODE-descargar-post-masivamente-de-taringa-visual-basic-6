REQUISITOS GENERALES DEL SISTEMA:

* Procesador de 500 megahercios (MHz) o superior
* Memoria: RAM de 256 megabytes (MB) o superior.
* Disco duro: 1,5 GB; una parte del espacio de disco se liberara tras la instalacion si el paquete de descarga original se quita de la unidad de disco duro.
* Unidad: Unidad de CD-ROM o DVD
* Pantalla: Monitor de 1024 por 768 de resolucion o superior
* Sistema operativo: Windows XP with Service Pack (SP) 3 (32-bit operating system (OS) only) or Windows Vista with SP1, Windows 7, Windows Server 2003 R2 with MSXML 6.0, Windows Server 2008, or later 32- or 64-bit OS.
* Otros: No tiene que sustituir el hardware que pueda ejecutar Office 2007, porque sera compatible con Office 2010.
