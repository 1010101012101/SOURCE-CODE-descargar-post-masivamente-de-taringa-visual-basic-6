Zend Studio 8.0.0 Professional Edition for Windows (Beta) | 350MB

Zend Studio entorno profesional de desarrollo y depuración de proyectos web. Una característica distintiva del programa es la capacidad de hacer la depuración remota y perfilado. Tenga en cuenta que para la depuración remota es necesario para instalar Zend Studio Server, que es un módulo de servidor. Como parte de Zend Studio como un paquete de opciones incluye PHP con una extensa lista de extensiones de pre-compilados. Si ya dispone de la versión pre-configurada de PHP, Zend Studio producto es fácil de integrar con él. El paquete también incluye soporte Zend Optimizer, es un módulo de servidor para ejecutar codificado con Zend Encoder y Zend SafeGuard Suite secuencias de comandos, así como algunos de sus excesos de velocidad. La séptima versión de Zend Studio agrega soporte para PHP 5.3, integración con Zend Framework y Zend servidor, código fuente, la edición y el programa de diversas mejoras de rendimiento.
