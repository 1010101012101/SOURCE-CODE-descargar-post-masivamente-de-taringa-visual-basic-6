Bueno para todos los que andan buscando el Microsoft Office 2010 con activador, aca les traigo mi aporte. Es una imagen en formato iso (719MB) y el activador aparte para descomprimirlo con WinRar - para los que por alguna razon no tienen este programa, vean mi otro Post: WINRAR EN ESPAÑOL + KEYGEN-
Continuemos, este Office 2010 (32bits) contiene:

* Microsoft Access 2010
* Microsoft Excel 2010
* Microsoft InfoPath 2010
* Microsoft OneNote 2010
* Microsoft Outlook 2010
* Microsoft PowerPoint 2010
* Microsoft Publisher 2010
* Microsoft SharePoint Workspace 2010
* Microsoft Word 2010

Bueno una vez descargada la iso, la pueden montar con Daemon Tools que lo pueden descargar en este link:
http://daemon-tools.softonic.com/descargar
con ese programa les quedara la imagen montada del disco de Micrsoft Office en una unidad, se van a la misma y le dan a autorun, reproduccion automatica, install, etc. Y va a comenzar el instalador, siguen las instrucciones como cualquier instalacion tipica.
bueno una vez ya instalado su Office 2010... lo que van a hacer es activarlo para tenerlo validado y sin molestias. Descargan el activador o si ya lo descargaron lo descomprimen y abren el archivo Office 2010 Toolkit... le dan click en donde dice EZ-Activator (Letras Verdes) y listo. ya lo tienen!!

Requisitos:

Equipo y procesador: Procesador de 500 MHz o más rápido

Memoria: 256 MB de RAM; mejor con 512 MB para las características de los gráficos, búsqueda instantánea de Outlook y ciertas funciones avanzadas.

Disco duro: 3,0 GB de espacio en disco disponible

Pantalla: Monitor con resolución 1024x576 o superior

Sistema operativo:

Windows XP (debe tener SP3) (32 bits)
Windows 7
Windows Vista con Service Pack (SP) 1
Windows Server 2003 R2 con MSXML 6.0 (solo Office de 32 bits)
Windows Server 2008 o SO de 32 o 64 bits posterior.
