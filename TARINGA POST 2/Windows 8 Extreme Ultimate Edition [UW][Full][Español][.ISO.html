Windows 8 xtreme un sistema basado en Windows 7 Final. No tiene ningún tipo de componentes retirados de la imagen original de Windows 7 así que no perderas ningúna compatiblilidad con el hardware o software que tienes. El sistema se instala en inglés por defecto pero incluye un MUI para pasarlo al español luego de la instalación.





Lo que incluye
* UAC Deshabilitado
* Tomar posesión
* Mostrar extensiones de los archivos
* Desfragmentación automática con Diskeeper
* Comando Nuevo archivo de clic derecho
* Habilitada la búsqueda en carpetas del sistema
* ClearType
* Windows le dirá exactamente lo que está haciendo cuando se está cerrando o se inicia
* Aumentado el rendimiento de la red
* Ajustes para IE8
* Nuevos temas Windows 7
* Botón de Inicio Nuevo
* Nuevo 3D Cursor
* Nuevos Fondos HD
* Nuevos iconos HD integrados en el sistema
* Nuevo Usuario
* Fotos
* Nuevos Sonidos de Windows
* Nueva pantalla de inicio
* SPTD
* Fx + Cursor Cursor 3D (separada opcional)

Requerimientos Minimos
* Procesador DC 2.0 GHz o Superior
* Memoría: 1 GB RAM
* Espacio en Disco Duro: 10GB de espacio libre
