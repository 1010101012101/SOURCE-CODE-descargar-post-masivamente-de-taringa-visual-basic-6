Descripcion:

La forma más rápida y más fácil de ejecutar la totalidad de sus programas favoritos! TurboLaunch es una herramienta totalmente configurable-barra de aplicaciones de estilo que se puede configurar para ejecutar muchos programas y realizar ciertas acciones pre-programados. Al igual que una barra de herramientas, las imágenes se utilizan para distinguir lo que es un botón está configurado para ello. TurboLaunch utiliza automáticamente el icono por defecto, pero usted tiene el control total y puede cambiar el icono de lo que quieras!

Los botones de TurboLaunch son fáciles de configurar! Arrastrando y soltando los programas, archivos, accesos directos o carpetas, inmediatamente puede configurar el botón para ejecutar el objeto caído. También puede configurar los botones de forma manual mediante la especificación de sus comandos, parámetros, carpeta de trabajo y estilo de ventana. Incluso tiene la posibilidad de asignar una tecla de acceso directo a cualquier botón, lo que le permite realizar con rapidez ese botón con un comando de teclado simple.

No sólo se puede configurar un botón para ejecutar un programa, pero también se puede configurar los botones para realizar ciertas acciones pre-programados. Al configurar un botón para activar el protector de pantalla, apagado de Windows, instalar un programa, reiniciar el sistema, o más, usted tiene un único clic a las cosas que normalmente tomar varias medidas para llevar a cabo!

También puede configurar un botón en TurboLaunch para establecer una conexión de red (como Internet o una conexión VPN). Esto hace que la conexión a su proveedor de servicios de Internet muy fácil, con características interesantes, tales como rellamada automática y Desconectar si está ya conectado. Asignar una tecla de acceso directo a la conexión de red para que pueda conectarse y desconectarse con sólo pulsar una tecla simple!

TurboLaunch es totalmente configurable. Usted puede ajustar el número de filas y columnas de botones que aparecen en la ventana TurboLaunch, e incluso organizarlos en grupos. Retire la barra de título, que TurboLaunch siempre estar al tanto de otras ventanas, y cambiar el tamaño de los botones más grandes o más pequeños para facilitar la visualización o para ahorrar espacio en la pantalla!

Después de usar TurboLaunch sólo por un corto período de tiempo, estamos seguros que usted encontrará TurboLaunch a ser una herramienta invaluable para el entorno de Windows!

TurboLaunch: Características principales:

• Una tecla de acceso directo se pueden asignar a un botón para poner en marcha muy rápida con sólo un comando de teclado rápido.
