WavePad software de edición de audio es un programa editor de sonido para Windows, Mac OS X y Pocket PC.

Este software de edición de audio te permite crear y editar audio, voz y otras grabaciones de audio. Cuando la edición de audio se puede cortar, copiar y pegar partes de grabaciones y, si es necesario, añadir efectos como eco, amplificación y reducción de ruido. WavePad es un editor de wav y editor de MP3, y soporta un número de otros formatos de archivo incluyendo vox, gsm, Real Audio, AU, AIF, flac, ogg y mucho más.

WavePad está diseñado para ser muy fácil e intuitivo de usar paquete de edición de audio. En cuestión de minutos usted será capaz de abrir o grabar un archivo y editarlo. Pero si te tomas el tiempo para explorar las otras características que encontrará muchas herramientas poderosas diseñadas con el ingeniero de sonido profesional en mente.
