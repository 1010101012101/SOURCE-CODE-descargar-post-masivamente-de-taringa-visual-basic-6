GIF Construction Set Profesional - Este programa está destinado a crear animaciones GIF. GIF Construction Set Profesional es capaz de crear animaciones para principiantes así como para usuarios más experimentados. La nueva herramienta llamada "Supercompresor" te permite minimizar la cantidad de animación GIF que necesites usando varios métodos. GIF Construction Set Profesional Puede abrir y guardar archivos AVI, así como utilizar diferentes paletas de colores.

* Exporte archivos GIF animados para MNG.
