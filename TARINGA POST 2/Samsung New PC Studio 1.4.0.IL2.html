¿Qué es el Nuevo Samsung PC Studio?
El Nuevo Samsung PC Studio es aplicación para PC la cual te hace fácil acceso a su dispositivo, puedes crear copias de seguridad y sincronizar el contenido con la PC. Administrar el su contenido personal y disfrutar de varios poderosos medios de comunicación de archivo herramienta de gestión que ofrece Samsung.

Características principales:

PIMS de Gestión
Editar, almacenar y sincronizar los contactos almacenados en tu teléfono móvil con el PC y el Outlook utilizando widget agenda. Y gestionar los datos PIM por Programador, a hacer, reproductores Memo fácilmente.

Media Manager
Gestionar, editar y reproduce contenidos como música, imágenes, archivos de vídeo en Media Manager libremente.
Todos los archivos de vídeo se puede convertir automáticamente (en formato y tamaño), aptos para el teléfono móvil conectado cuando se transfieren los archivos.

Actualización de Firmware
Nueva PC Studio ofrece el software (firmware) de actualización de teléfono móvil. Si el teléfono móvil conectado las necesidades de actualización, fuentes de energía nuclear y notificar a la última actualización de software disponible para su teléfono.

Requerimientos Sistema
SO: Windows 2000 (Service Pack 4), Windows XP (Service Pack 2), Windows Vista, Windows 7
CPU: Pentium 600 MHz o superior (se recomienda)
Cantidad mínima de memoria (RAM) Tamaño: 512 MB
Espacio libre en disco: 200 MB
Los métodos de conexión: cable USB o Bluetooth
