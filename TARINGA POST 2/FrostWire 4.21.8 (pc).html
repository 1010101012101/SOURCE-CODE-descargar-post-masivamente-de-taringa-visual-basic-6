Ya tenemos disponible una nueva versión de este cliente P2P multiplataforma, alternativo al desaparecido Limewire: FrostWire 4.21.8. FrostWire es un programa gratuito, que tiene como característica principal, al igual que el comentado LimeWire, utilizar las redes Gnutella y BitTorrent para buscar todo tipo de ficheros, (música, películas, documentos imágenes, etc). FrostWire es un programa multiplataforma, que está disponible para los sistemas operativos Windows, Linux y también para Mac OS X y está escrito en Java.

En FrostWire 4.21.8 se corrige un error al congelarse el programa cuando se muestra el asistente de configuración, también se corrige un problema con las carpetas de guardado de las descargas y desactiva compartir. También se añaden opciones para compartir en el botón de buscar cuando la búsqueda se ha iniciado y se añaden opciones en el menú de ayuda para aprender a compartir archivos.
