Windows XP Sp3 uE v2 es básicamente un Windows XP con Service Pack 3, con actualizaciones integradas, desatendido, con algunos componentes dispensables eliminados y que además lleva un par de aplicaciones básicas muy útiles, también de forma desatendida.

¿Que tiene integrado?
Esta versión lleva por supuesto Service Pack 3, Internet Explorer 8, Windows Media Player 11 y DirectX 9c integradas, todas actualizadas hasta Agosto 2010.
Lleva parches de SFC, limite de conexiones aumentado, temas no oficiales, WGA, OGA, etc.

Prácticamente todos los equipos nuevos ahora vienen con discos duros SATA y probablemente controladores SATA que Windows XP no soporta de forma nativa. Normalmente, para poder instalar hay que tener los controladores en un disquete y presionar F6 al principio de la instalación…
Pero todo eso no hace falta, porque Windows XP Sp3 uE v2 lleva un amplio paquete de controladores SATA para poder instalarlo cómodamente incluso en RAID.
Novedades

* Tal y como lo pone arriba, esta versión es fiel a la filosofía Windows uE, y esa es: “Estabilidad, rapidez y calidad.” No interesa un equipo que vaya rápido pero mal, por eso solo se han quitado componentes dispensables. Lleva todas las actualizaciones post SP3 integradas para una mayor seguridad y un mejor funcionamiento.
* Las aplicaciones han sido actualizadas en totalidad, algunas desde cero, incluyendo así muchas actualizaciones, como es el caso de .NET Framework.
* Esta versión es aún más desatendida. El proceso de instalación se paraba en dos puntos antes:

- Antes de WPI para poder cambiar la resolución del monitor y aplicar el tema.
- Después de WPI, para establecer la contraseña y el nombre del equipo.
Por supuesto, ambos puntos son cruciales, pero ahora la selección de la resolución lo hace de forma automáti... ca cambiando a 1024×768 (ó 800×600 si no soporta más), y luego siguiendo con WPI. El asistente para la contraseña y el nombre del equipo está al final del todo, cuando están las aplicaciones instaladas y solo hace falta reiniciar. Prácticamente ahora la instalación es 95% desatendida.

* Después de seleccionar la partición, el equipo instala Windows, las aplicaciones predeterminadas de WPI y termina con el asistente para contraseña y nombre del equipo. Como último paso, reinicia de forma automática y todo está listo.
* Se ha tenido mucho más en cuenta los usuarios creados después, ademas del usuario principal Administrador. Se ha comprobado cada aplicación, que este configurada en español y que los accesos directos funcionen.
* Quitado el archivo cmdow.exe para ocultar ventanas de instalación, que tantos usuarios sin conocimientos reportaban como virus. El proyecto ha sido creado en un entorno limpio, además, la ISO ha sido comprobada con varios antiviruses, confirmando que la imagen ISO es libre de cualquier archivo dañino.

Aplicaciones
En cuanto a aplicaciones, la lista ha sido muy simplificada. Los motivos son varios:

* Actualizaciones del sistema han incrementado el tamaño de Windows.
* Las aplicaciones principales no solo han cambiado la versión, pero el tamaño también, aumentando por supuesto.
* Algunas aplicaciones se han quedado obsoletas o no eran tan utilizadas por la mayoría de la gente.
* Otro ejemplo podría ser que ya no están las aplicaciones QuickTime Alternative, Real Alternative ni PowerDVD. Las funciones de dichas aplicaciones las realiza el paquete K-Lite Codec Pack en su versión Full.

Notas Finales
[Instalación]
Corta descripción de la instalación:
-> Iniciar instalación
-> Seleccionar partición / disco y formatear
-> Copiar archivos
-> Reinicia e instala Windows
-> Reinicia y configura la resolución
-> Configura el paquete de controladores SATA y arranca WPI
-> Instala aplicaciones WPI
-> Aplica configuración Windows
-> Limpia archivos temporales
-> Inicia Asistente para establecer contraseña y cambiar el nombre del equipo
-> Reinicia

* Siempre formatear con una instalación de Windows. Es lo mejor para el equipo. También se recomienda hacer la instalación en la partición C:.
* Dejar el disco en la unidad hasta que la instalación este lista.
* Aunque se puede arrancar WPI en otros equipos o después de la instalación, es probable que de algún fallo. El WPI está hecho para que funcione cuando el arranca SOLO en la instalación de Windows XP Sp3 uE v2.
* Esta versión de Windows está optimizada, aun así, se recomienda instalar solamente las aplicaciones necesarias. Leer descripción en WPI si hay dudas.
* Una vez la instalación finalizada, esperar un poco a que termine de limpiar los archivos temporales y salga el asistente – esto puede tardar hasta dos minutos dependiendo de la configuración del equipo.
* Después de la creación de nuevos usuarios y el primer login, hay que reiniciar para que se configure todo.
* La descarga es un archivo ISO – No descomprimir con WinRAR.
