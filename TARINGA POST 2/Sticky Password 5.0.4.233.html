Completo gestor de contraseñas con sistema de inserción automática

Sticky Password es un completo administrador de contraseñas con el que podrás almacenar todas tus claves en un lugar seguro, utilizarlas cuando sea necesario con un simple clic y además te ahorrarás el tener que memorizarlas todas por separado.
El programa guarda todas las contraseñas en una base de datos totalmente protegida mediante potentes algoritmos de encriptación (AES/Rijndael, Blowfish, Gost, SCOP, Twofish y más, hasta 8192 bits) y una clave “maestra”, que será además la única que tendrás que recordar para acceder a tus contraseñas.

Pero además de guardar las claves a buen recaudo, Sticky Password te permite insertarlas donde sea necesario (páginas web, servidores FTP, programas de mensajería instantánea, etc.) de forma muy cómoda, gracias a un botón que aparece en la barra de título de la ventana.
El programa tiene soporte para múltiples nombres de usuario y claves dentro de una misma cuenta, incluye un generador de contraseñas al azar y realiza periódicamente copias de seguridad de la base de datos para no perder ni una clave.
