Hoy les voy a enseñar como instalar Windows xp SP3 desde un disco duro local.

Es muy fácil, tanto para hacer una actualización como para reinstalar todo el Windows.

Lo que van a necesitar son las siguientes cosas:

1.- Ultraiso: http://www.mediafire.com/?twtmdmjymwt (3.17mb, con Serial incluido)

2.- Un ISO o NRG con los archivos de instalación de Windows xp SP3: http://www.fileserve.com/file/kMtMwv9 (590.80mb, con claves incluidas)

3.- Clave: M8J3M-QRKTX-HT48F-R8CDV-4V27J

4.- Un disco duro libre (o alguna particion), con formato NTFS y con por lo menos 900mb libres.

Bien, los pasos son los siguientes:
1 Procedemos a abrir el instalador de windows xp con el Ultraiso.

2 Extraemos los archivos a la unidad de disco duro libre.

3 Ejecutamos "Setup.exe".

4 Iniciamos la instalación de Windows.

Esta es una alternativa a los métodos clásicos de instalación (CD - USB) y créanme: Es muy segura y funciona excelente.
