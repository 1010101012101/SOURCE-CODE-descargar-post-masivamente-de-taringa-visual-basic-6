Dando vueltas a la red encontre el que para mi es el mejor tema de Windows 7 que existe: Es el tema SiCo. Es una especie de Aero mas estilizado, con mas brillo, mejores acabados, un menú inicio redefinido mas moderno, excelentes WallPapers, un Orb minimalista muy bueno, y combinacion excelente con distintos colores.

INSTALACION:

1° Descargas el Tema: http://www.fileserve.com/file/6Yjuhqh

2° Si no tienes Tune Up Descargalo: http://www.taringa.net/posts/downloads/8141014/TuneUp-2011-en-espanol-por-megaupload.html (este Post no es mio, pero ya han subido tantas veces tune up que me dio pereza subirlo de nuevo)

3° Abre Tune Up/personalizar Windows/Estilo Visual

4° Dale añadir/Buscar estilo visual y busca en la carpeta que descargaste Sico Big.msstyles o Sico Small.Msstyles (la diferencia entre uno y otro es el tamaño del reloj)

5°Una vez abierto, seleccionalo en la lista y en la parte inferior izquierda dale aplicar. Despues de unos segundos Tendras el tema Aplicado

6° Los Wallpaper los instalas manualmente dandole clic derecho/Establecer como fondo de escritorio (sSon 4 muy buenos WallPaper Abstractos)

7° Si haz notado aun falta el Orb (pido disculpas por haberlos subido por aparte, estoy en un internet demasiado malo y no pude subir las dos cosas en el mismo link) Para descargar el Orb, dale clic aqui: http://www.fileserve.com/file/8g75ksK

8° Abre el Windows 7 Start button Changer como administrador, dale en select y change start button

9° Selecciona el Orb que viene en la carpeta descargado

10° Tienes tu windows totalmente personalizado pantalla pack theme tema personalizar tuneup orb abstracto imagenes Wallpaper fondo computador
