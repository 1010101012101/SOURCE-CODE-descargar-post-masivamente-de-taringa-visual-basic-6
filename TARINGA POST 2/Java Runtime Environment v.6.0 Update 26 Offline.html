Java es un lenguaje de programación y la primera plataforma informática creada por Sun Microsystems en 1995. Es la tecnología subyacente que permite el uso de programas punteros, como herramientas, juegos y aplicaciones de negocios. Java se ejecuta en más de 850 millones de ordenadores personales de todo el mundo y en miles de millones de dispositivos, como dispositivos móviles y aparatos de televisión.

Java Runtime Environment es la máquina virtual de Java más básica. Sin ella, ningún programa Java lograría arrancar, y lo mismo se aplica a los applets de las páginas web. Es un componente tan imprescindible como .NET o las librerías DirectX.

Existe un gran número de aplicaciones y sitios Web que no funcionan a menos que Java esté instalado, y muchas más que se crean a diario. Java es rápido, seguro y fiable. De portátiles a centros de datos, de consolas de juegos a superequipos científicos, de teléfonos móviles a Internet, Java está en todas partes.

Java le permite jugar en línea, participar en sesiones de chat con internautas de todo el mundo, calcular los intereses de una hipoteca y ver imágenes en tres dimensiones, entre otras muchas aplicaciones. Es también esencial para las aplicaciones de intranet y otras soluciones de comercio electrónico que constituyen la base informática de las empresas.

El instalador de Java Runtime Environment - o JRE - se encarga de integrar Java con el sistema y los principales navegadores (como Internet Explorer o Firefox). Sólo tienes que pulsar Siguiente hasta que la instalación de Java Runtime Environment se haya completado.

Si necesitaras configurar Java Runtime Environment, encontrarás, en el Panel de Control, un icono dedicado exclusivamente a Java.
