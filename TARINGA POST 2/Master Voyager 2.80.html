crear una contraseña protegida de DVD / CD y discos USB Flash Drives

Maestro Voyager fue especialmente diseñada para crear una contraseña protegida de DVD / CD y discos USB Flash Drives (Pendrives). Protegidos los medios de comunicación es totalmente autónomo y no necesita ningún software especial instalado en la computadora que se opera. Puede crear discos protegidos con almacenamiento en el equipo de instalación y "jugar" o abrir archivos en cualquier otro ordenador (requiere Windows 7, Windows Vista o Windows XP).





• Descifrado transparente. No es necesario para descifrar los datos de DVD / CD / USB o copiar archivos desde CD / USB al disco duro. Basta con introducir una contraseña y trabajar con encriptado USB o CD . Abrir documentos cifrados directamente desde el CD / USB e incluso poner en marcha programas o reproducir películas. Maestro Voyager utiliza el exclusivo sobre la marcha-la tecnología de descodificación desarrollado exclusivamente para CryptoExpert Professional .
• Confidencialidad de datos ultra segura. Maestro Voyager utiliza el cifrado fuerte de todos los datos que utiliza AES de 256-bit de protección. AES es el Federal Information Processing Standard (FIPS) que se utiliza para proteger la información más sensible del gobierno estadounidense y organizaciones comerciales, incluidos los bancos. Lea más en nist.gov
• Integrados de CD / DVD módulo quema para crear DVD o CD encriptados . Maestro Voyager apoya CDR / CDRW / DVDR / DVDRW tipos de disco. Sólo tiene que seleccionar los archivos / carpetas que necesita para almacenar de forma segura en cualquier medio de almacenamiento principal y ese señor Voyager hacen el resto.
• No hay necesidad de receptores disco para funcionar Maestro Voyager. Si envía un disco o una memoria USB a través del correo electrónico a un tercero, que no es necesario para operar Maestro Voyager, que simplemente necesita la contraseña para desbloquear los datos protegidos y la información. El tercer usuario simplemente tiene que insertar el medio de almacenamiento a su ordenador e introduzca la contraseña para trabajar como lo harían con cualquier otro CD o dispositivo USB. El único requisito es tener instalado Windows 7/Vista/XP (de 32 bits y 64 bits son compatibles). No hay ninguna descarga de software son necesarios y no hay pérdida de tiempo se reinicia o configuraciones de los equipos que preocuparse, el proceso es sencillo y automatizado.
