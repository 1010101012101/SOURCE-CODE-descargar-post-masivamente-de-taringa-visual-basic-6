Longtion Application Builder es una herramienta para diseñar y desarrollar aplicaciones de base de datos, aplicaciones Web y la ejecución automática de CD del menú sin tener que escribir ningún código.

Y usted no tiene que ser un programador de software para hacerlo. Gracias a su intuitivo entorno de desarrollo integrado (IDE) y los objetos de arrastrar y soltar, es fácil de aprender y fácil de usar.

No requiere conocimientos de programación alguna! Usted se sorprenderá con la funcionalidad de gran alcance y atractivo visual elegante que Longtion Application Builder pone a su ratón clic en la yema del dedo. Diseñar y distribuir el software con unos pocos clics del ratón.

Con Longtion Application Builder usted puede construir fácilmente aplicaciones Web exactamente la misma manera que crear aplicaciones de bases de datos normales. Se trata de una nueva y revolucionaria forma de crear aplicaciones basadas en web. La aplicación web independiente es un servidor web que se ejecutan sin dependencias de cualquier servidor web (IIS o Apache).
