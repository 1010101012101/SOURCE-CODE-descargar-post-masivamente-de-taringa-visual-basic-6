Advanced Uninstaller es una aplicación para Windows que tiene como función principal la desinstalación de los programas, carpetas y archivos que no se quitaron con el desinstalador estándar de Windows.
Además el programa realiza una serie de actividades de limpieza, haciendo que tu ordenador sea más seguro.

También puedes eliminar de forma segura el historial de páginas de internet, las direcciones web que has tecleado, las cookies y la lista con archivos abiertos recientemente para más de 100 aplicaciones, incluyendo el Microsoft
Office, el reproductor de Windows Media, ICQ, MSN, WinZip, RealPlayer o el Kazaa.
Asimismo podrás ajustar los valores de configuración de Internet Explorer, gestionar y desinstalar las fuentes, desinstalar los subprogramas del panel de control y realizar otras tareas de manera eficiente.
