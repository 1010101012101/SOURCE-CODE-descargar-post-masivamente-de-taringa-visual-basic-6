TEMAS

Los temas se dividen en dos clases, oficiales y no oficiales.

Temas oficiales de Windows 7



Para instalar los temas oficiales basta con descargarlos y abrir el archivo descargado. Automaticamente se nos instalará un wallpaper, un color para la barra de tareas y las ventanas, sonidos e íconos.

Temas no oficiales de Windows 7



A la hora de hablar de la instalación de estos se complica un poco, aunque no es difícil instalarlos. Para proceder a instalarlos debemos de aplicar el UxTheme Patch, tenemos dos formas para ello, la forma manual o con UxTheme Patcher.

CURSORES

Los cursores son un poco menos comunes, y por ende mas difíciles de encontrar. Un buen sitio para buscarlos es deviantART.



Para la instalación de estos, solo tenemos que descargarnos los íconos y descomprimirlos. Ahora damos con el botón derecho en el escritorio y gt; personalización y se nos abrirá una ventana como esta. Vamos a cambiar el cursor (el cursor que usaré para este tutorial lo podéis descargar aquí)



Al darle, nos saldrá la ventana de los iconos que tiene Windows 7 por defecto. Seleccionamos el primer cursor y le damos a Examinar



Seleccionamos la ubicación del cursor que queremos agregar.



Al abrirlo, nos aparecerá en el lugar que antes estaba el anterior cursor. Ahora, repetimos el proceso con los iconos de ayuda, de cargar, etc. Luego le damos a aplicar y tendremos todo hecho.
