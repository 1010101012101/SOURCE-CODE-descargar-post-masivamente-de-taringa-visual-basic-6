INFORMACIÓN:

VirtualBox - programa para ejecutarse en el mismo equipo varios sistemas operativos virtuales. Esta es una herramienta útil y funcional para la virtualización, kahk a empresas y usuarios domésticos. El programa es compatible con muchos sistemas operativos diferentes invitados de la familia de Windows (NT/2000/XP/2003/Vista/7), así como DOS, Linux y OpenBSD. Al igual que en otras aplicaciones similares, existe la posibilidad de iniciar y detener el sistema operativo invitado, configurar el uso de los recursos informáticos, el intercambio de datos entre el host y el sistema operativo invitado, realizar capturas de pantalla de los sistemas virtuales, y mucho más.
Características principales:
“Soporte para USB 2.0, cuando las máquinas host del dispositivo están disponibles para los clientes de OS
“Built-in RDP-servidor, cliente y soporte para dispositivos USB sobre protocolo RDP
“El soporte experimental para un disco duro VMDK / VMware
“Apoyo a iSCSI
“La virtualización de audio
“Apoyo a los distintos tipos de redes (NAT, red de acogida a través de puente, Interior)
“Apoyo para el árbol de estado guardado de la máquina virtual (instantáneas), que se pueden deshacer de cualquier Estado invitado
“Apoyo a las carpetas compartidas para compartir fácilmente archivos entre el host y los sistemas de invitados
“Compatibilidad con múltiples plataformas
“La modularidad
“Y mucho más …
