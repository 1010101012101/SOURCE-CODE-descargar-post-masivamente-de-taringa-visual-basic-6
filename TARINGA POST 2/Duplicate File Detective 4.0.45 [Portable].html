Duplicate File Detective se trata de un programa capaz de detectar todos los archivos duplicados en tu disco duro, y procesarlos mediante diversas acciones: moviéndolos de lugar, archivándolos en un ZIP, o directamente borrándolos del disco.

El programa utiliza un potente motor de escaneo multihilo con soporte para múltiples rutas simultáneas, tanto locales como de red, así como opciones de fitlrado para incluir o excluir determinados tipos dearchivos.

Los resultados se presentan en una lista sobre la cual podrás marcar manualmente los objetos que quieras procesar. Características adicionales de Duplicate File Detective: soporte para línea de comandos y calculador de hashes con soporte para CRC32, Adler32, MD5 y SHA1.
