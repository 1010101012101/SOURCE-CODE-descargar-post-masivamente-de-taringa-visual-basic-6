Pestañas en la parte superior

De forma predeterminada (de momento en Windows, mientras que en Linux y Mac se puede cambiar manualmente) las pestañas se ubican ahora en la parte superior para darlas más prioridad y obtener una navegación más eficiente e intuitiva.

Mejoras de privacidad

Se han implementado diversos cambios para mejorar la privacidad de los usuarios, como que el selector de css :visited en sitios maliciosos no pueda intentar obtener información del histórico de navegación.

Otros cambios

Mejoras en el API utilizado por la extensiones y avances en JetPack lo que permitirá que usuarios menos expertos puedan crear sus propias extensiones.

Unión de los botones de recargar y parar en uno que los combina.

La barra de marcadores se ha transformado en un botón de marcadores, aunque si se quiere puede revertirse al formato anterior.

Protección frente a cuelgues de los plugins de Adobe Flash, Apple Quicktime o Microsoft Silverlight, tanto en Windows, Linux y Mac.

Implementación parcial de las transiciones CSS (en).
Implementación total de WebGL (en) aunque de forma predeterminada está desactivado.

Dibujado nativo de animaciones para plugins en Mac, lo que hará que ahora los plugins dibujen de una manera más rápida y eficiente.

Uso experimental en Windows del dibujado mediante Direct2D, aunque por defecto se encuentra desactivado.

Los desarrolladores web podrán usar websocket (en) lo que reducirá la complejidad de comunicación entre cliente y servidor en ambas direcciones.
Los desarrolladores web podrán cambiar la dirección en la barra de direcciones sin que se recargue la página utilizando la API del historial HTML (en).

Dibujado más fluido gracias al uso del lazy frame construction (en).

El volver a páginas anteriores es ahora asíncrono, lo que proporciona una mejor respuesta durante la carga de la página.

Nuevo ‘parser (en)‘ para HTML5.

Manejo de nuevos controles de html5.

Web Console, es una herramienta aun experimental para poder analizar los actuales sitios web.

Base de datos integrada, ahora los desarrolladores web cuentan con una potente base de datos integrada que hará que sitios como GMail o GDocs funcionen mucho más rápidos e incluso puedan funcionar sin necesidad de estar todo el tiempo conectados a Internet.
