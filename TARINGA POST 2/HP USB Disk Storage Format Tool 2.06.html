MODO DE EMPLEO:
1. Conecte el dispositivo USB a un puerto USB disponible.

2. Descargar el SoftPaq en un directorio temporal en su disco duro.
El archivo descargado es un ejecutable autoextraíble con un nombre de archivo basado en
el número de SoftPaq anterior.

3. Ejecute el archivo EXE descargado SoftPaq para descomprimir archivos de programa. esto
silencio a instalar la utilidad format. Durante la instalación, si se le pide
para sobrescribir los archivos, responda Sí a todo para continuar instalar el software.

4. Cuando la instalación finalice, los archivos de instalación y temporales
directorios se pueden borrar.
