Otro Activador para Windows 7, al igual que 7 Loader de Orbit y Hazar, este nuevo activador hecho por Napalum promete activar cualquier windows 7, así como lo lee cualquier windows 7 (ultimate, professional, premium, etc), esta es una de las características carentes en el activador de orbit y hazar.
Se comenta que este nuevo activador emplea las Keys de productos OEM para lograr su propósito, pero tener mucho cuidado que a un largo plazo estas keys podrían ser bloqueadas.
