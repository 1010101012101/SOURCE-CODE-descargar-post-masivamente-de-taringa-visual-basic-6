Este tutorial es mi traducción de un tutorial sacado de la web wiki Gitbrew. El motivo de este tutorial es hacer el proceso de downgrade (bajar de versión de FW) de 3.55 o menor hasta el FW que se quiera instalar en tu PS3 (ya sea 3.41, 3.15 etc). A continuación el proceso para hacer el downgrade y más abajo están los enlaces de las descargas necesarias. También necesitas el archivo del FW Original al que deseas bajar; el (PUP) del 3.15 o 3.41, etc).

IMPORTANTE:
Cita: Haz un backup de tus archivos, porque éste proceso puede formatear tu PS3 y perderás las descargas de PSN etc.


Instala el CFW355-OTHEROS++-SPECIAL.pup (No importa desde que versión lo haces [3.41, 3.50 etc etc]).Instala el paquete qa_flag_extra.pkg.Ejecuta el qa_flag.Si escuchas un silbido, continua. Si no lo escuchaste, vuelve a comenzar.Reinicia.Entra al recovery menu y actualiza tu PS3 con el Firmware que quieras (3.15, 3.41 etc)Ahora será instalado.
