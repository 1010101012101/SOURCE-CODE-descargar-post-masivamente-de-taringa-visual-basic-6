Muchos han querido grabar sus archivos de música, videos o documentos. Pero no encuentra la solución para poder hacerlo, y es por eso que existe PowerISO. Una aplicación con muchos detalles

PowerISO es un potente CD / DVD archivo de imagen herramienta de procesamiento, que le permite abrir, extraer, crear, editar, comprimir, encriptar, dividir y convertir archivos ISO, y montar estos archivos con la impulsión virtual interna. Puede procesar casi todos los archivos de imagen de CD-ROM incluyendo ISO y BIN.

Características principales:

Soporta casi todos los CD / DVD-ROM en formato de archivo de imagen (ISO, BIN, NRG, CDI, DAA y así sucesivamente).
Nota : DAA (Direct-Access-Archive) es un formato avanzado de archivo de imagen, que apoya algunas características avanzadas, tales como compresión, protección con contraseña, y dividir a varios volúmenes. Puede ser manejado directamente al igual que otros formatos, tales como ISO, BIN, …

Soporte de 32 bits y 64 bits de Windows.

Abrir y extraer archivos ISO.

Crea archivos ISO desde el disco duro o CD / DVD-ROM.

Editar un archivo ISO directamente.

Convertir archivos de imagen entre ISO / BIN y otros formatos. PowerISO puede convertir casi todos los formatos de archivo de imagen a formato ISO estándar.

Hacer de arranque de archivos ISO, obtener información de arranque de archivos ISO.

Haga disquete archivo de imagen de disco.

Optimización de archivos para ahorrar espacio en disco al guardar archivo ISO.

Monte el archivo ISO con la impulsión virtual interna. No es necesario instalar ningún otro software de unidad virtual.


Puede ser utilizado con facilidad. PowerISO apoya integración shell, menú contextual, arrastrar y soltar, portapapeles copiar pegar…
