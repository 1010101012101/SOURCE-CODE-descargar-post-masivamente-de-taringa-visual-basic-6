Skype es un software gratuito que permite realizar llamadas de voz y video con otros usuarios de este
software sin costo alguno, las llamadas se realizan a través de una conexión a Internet, permite además la
transferencia de archivos de gran tamaño, conferencias con otros usuarios del software, chats; todas las
llamadas y los archivos son encriptados para que no puedan ser interceptados por otros usuarios. Este
software también permite realizar llamadas a teléfonos convencionales y móviles con el pago de una mínima
tarifa.

Más allá de la voz, está el vídeo: Skype 5 admite ver hasta diez personas en una sola sesión de vídeo-conferencia. Cada recuadro de vídeo se muestra sobre un elegante fondo azulado. Invitar a otras personas a la vídeoconferencia de Skype es tan sencillo como hacer clic sobre “Agregar personas” y elegir el contacto.

Skype es el nuevo teléfono. La quinta versión, con videoconferencia de grupo, integración con redes sociales y un aspecto aún más agradable, pone este programa a años luz de sus competidores.



Interfaz Grafica:

1) Usuario: Perfil de usuario, aquí puede poner un mensaje para que todos sus contactos lo lean cuando inicie sesión, puede cambiar su estado a: conectado, ocupado, desconectado, invisible.

2) Lista de contactos: En esta lista se encuentran todos los contactos del usuario, se pueden mostrar solamente aquellos que están conectados o se pueden mostrar todos los contactos.

3) Pestañas de funciones adicionales: además de poder realizar llamadas gratuitas con otros usuarios de Skype, el usuario puede hacer llamadas a teléfonos convencionales o móviles cambiando de pestaña a “Marcar” ( (este servicio requiere el pago de una tarifa anual o trimestral). La tercera pestaña “Registro”, se guarda un registro de todas las llamadas realizadas.

4) Botones de Herramientas: Estos botones permiten controlar toda la funcionalidad básica del software Skype, añadir contacto, buscar contacto, conferencia (llamada con 4 usuarios simultáneamente), Chat (con otros 28 usuarios), enviar un archivo o modificar el perfil de usuario.

5) Llamar/Colgar: Estos dos botones se utilizan para llamar al contacto seleccionado o para terminar la llamada.
