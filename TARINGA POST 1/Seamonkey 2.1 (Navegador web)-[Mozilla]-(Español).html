SeaMonkey es un producto de la fundación Mozilla que reúne bajo el mismo brazo un navegador web con pestañas, un cliente de correo electrónico y grupos de noticias, un cliente IRC y un editor HTML.

El resultado es una completa y práctica suite que incluye las aplicaciones necesarias para comenzar a navegar a través de Internet. El nombre anterior de esta herramienta fue 'Mozilla Application Suite'.

El navegador web incluido en SeaMonkey presenta algunas diferencias destacables con respecto a Firefox, entre ellas: un traductor de páginas integrado; gestor cookies, imágenes, pop ups y formularios; y, por supuesto, varias herramientas de desarrollo web: consola de errores, consola JAVA, inspector DOM y depurador de Javascript.

El cliente de correo permite componer mensajes de correo e incluir toda clase de etiquetas HTML en el cuerpo textual. Además, integra una agenda de contactos. ChatZilla! es el nombre del cliente IRC, que te permitirá mantener conversaciones de chat en cualquier servidor de este protocolo.


Fanáticos de la clásica Suite de Internet de Mozilla alegraros. La actualización mayor para SeaMonkey ha sido publicada incluyendo un gran número de mejoras significativas. Aquí te comento algunas de ellas:

Recuperación de sesión: Si SeaMonkey se cierra o cuelga de manera inesperada, tu sesión se puede restaurar.
Restaurar Ventana, ve a Archivo/Ventanas recientemente cerradas para recuperar una ventana cerrada.
Barras de herramientas personalizables, botón derecho sobre la barra de herramientas y elige Personaliza… para añadir o quitar botones.
Resumen de descargas entre sesiones.
Soporte para hilos RSS y Atom en el módulo de correo
Correos que se abren en nuevas pestañas en vez de nuevas ventanas para una mejor lectura de tus correos.
Remodelación de los administradores de descargas, formularios y contraseñas.
Actualización del tema para Mac OS X.
SeaMonkey 2 está disponible en 18 idiomas para Linux, Mac OS X y Windows.
