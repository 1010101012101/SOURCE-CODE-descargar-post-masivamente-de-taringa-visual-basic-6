¿Que Es WINDOWS EXTREME GAMING?

windows extreme gaming es un sistema operativo especializado para juegos











Tweaks, etc del suricaos removidos (los problemas debido a procesos mal configurados, etc, que podian llegar a causar problemas al instalar alguna aplicacion o driver, debido al suricataos, ya no estan, no tendras ningun problema de este tipo, y si los tenes busca en google, pues sera un problema especifico de tu driver en XP, no del WinEGE2. Seguro habra solucion).

No hay que activar/desactivar ningun proceso manualmente, si queres internet usa el net fix, si queres red usa el red fix.

El Red/Internet fixer fue separado, red por un lado internet por el otro, para que se ejecuten solo los procesos necesarios, ejecutando ambos fixes se obtienen las dos cosas.

Se instala el .NET Framework 1.1+sp1 (+hotfix) y el 2.0+sp1, en el proceso de instalacion del WinEGE (no lo notaras… exepto porque tardara mas en instalarse). Tambien se instalan las librerias vc08 redist 2005.Cualquier problema con los programas del menu booster fueron resueltos… (Live Messenger, Safari…).

Añadido pack de drivers chipset y cpu (amd).

Algunos juegos podian llegar a instalarse en ingles si eran multi-lenguaje, eso ya no pasa…

Sera posible instalar y usar la impresora, usando el ”printer fixer”, pese a que ningun juego lo requiere.

Ecualizador Realtek – Placa de Audio HD (archivo HHCTRL.OCX perdido) – Ya resuelto en el WinEGE2 gracias a autoesculeascano

Arreglado el problema con las librerias wm (wm*.dll) perdidas, gracias a kraven234 (tester).Ademas ahora podran disfrutar del windows media 9.

Agregado manual para crear particiones desde Windows Vista, mas dos metodos para hacer el Dual Boot (tener Vista+XP).

Agregado el Red-Basic Fix, podes probarlo, si tu red-local aun no funciona como quieres usa el Red-Full Fix.

Agregado Net-Basic Fix, podes probarlo, si tu conexion no funciona como deberia usa el Net-Full Fix. Nota: Wi-fi, Wireless, Wlan, conexioninalambrica o como quieran llamarlo solo funciona con el full fix.

Menu-apps (Mini-Booster), contiene solo los fix automaticos, se podra utilizar desde el escritorio (por defecto).


Menu Booster (dentro del ISO/CD) se incluye:

• DirectX 9.0c Redist Junio 2008
• Live Messenger
• Xfire 1.79
• TeamSpeak 2 Client
• VlSTABoot PRO 3.3
• Hamachi
• RivaTuner 2.09
• Daemon Tools 4.12 Lite
• Firefox 3 Portable
• CPU-Z
• Posibilidad de elegir el cache de 2do nivel entre 256KB hasta 64MB. (preparado para
procesadores Intel de 8 Nucleos)
• EGE Booster: Permite devolver el estado de los procesos a como estaban en el segundo reinicio de sistema (por lo que si tocaste algo, hay forma de revertirlo).
• Net-Full Fix: Habilita TODOS los procesos necesarios para crear una conexion a internet ADSL, Wireless o que requiera nombre de usuario o password. Cuando ya no necesites estar
conectado a la net, puedes usar el EGE Booster.
• Net-Basic Fix: Habilita los procesos BASICOS necesarios para una conexion ADSL simple (wireless, wi-fi solo con el full fix). Cuando ya no necesites estar conectado a la net, puedes usar el EGE Booster.
• Red-Full Fix: Habilita TODOS los procesos necesarios para terner la PC en red sin problemas. Cuando ya no necesites estar conectado a la red, puedes usar el EGE Booster.
• Red-Basic Fix: Habilita los procesos BASICOS necesarios para terner la PC en red sin problemas. Cuando ya no necesites estar conectado a la red, puedes usar el EGE Booster.

En cuanto al menu ”Menu-apps” (Mini-Booster) (en el escritorio) se incluye:

• EGE Booster, Net-Full, Net-Basic, Red-Full, Red-Basic.

• Printer Fix: Habilita el uso de la impresora.

Este SO esta pensado solo para jugar, si estas pensando usarlo para realizar tareas diarias, porque vista te funciona lento… olvidalo, mejor baja el win 2000, este windows no es para PC de bajo recursos (aunque funcione perfectamente en una Pentium 2).
El windows que todo gamer quiere para poder jugar juegos con un sistema operativo bastante liviano.

Esta hecho no solo para que use menos RAM (ese es el punto menos importante), esta hecho para concentrar todo el poder de tu procesador y demas hardware en el juego que estes corriendo, debido a numerosos tweaks de registro, componentes que se podian estar ejecutando y fueron removidos, menos servicios ejecutandose, etc.

Lo recomendado es que hagas dos o mas particiones en tu Disco Duro, en una instales el WinEGE2 para los juegos y en otra el Windows Vista (o XP) para las tareas y demas programas que usas diariamente.

La particion del WinEGE2 tiene que tener un minimo de 10GB, pues aunque ocupe poco, todo windows necesita espacio libre en su particion para funcionar correctamente (de lo contrario la velocidad se vera muy afectada), debido a la memoria virtual, etc…

• Es XP
• Es de 32BITS (X86)
• Es solo para jugar, no sirve para hacer tareas diarias.
• Van a ver que todas las herramientas necesarias para hacer un Dual Boot vienen incluidas, y da igual instalar EGE o VlSTA o XP primero o segundo (VlSTABootPRO+Manual incluido)
• Que necesite menos de 40MB de RAM (31MB sin explorer) no significa que sea para maquinas de bajos recursos, esta hecho para jugar lo mas nuevo, Crysis, Flatout Carnage y todo eso, incluso el cache de segundo nivel se puede configurar para que soporte procesadores de 64MB (Los Core2Duo tienen 4MB u 8MB). Por supuesto que en la maquinas viejas tambin se puede usar, pero si usas la PC para jugar es poco probable que tengas un Pentium2 (aunque funciona perfecto)
• Se puede instalar en cualquier particion.
• Se puede instalar en discos IDE y SATA sin problema.
