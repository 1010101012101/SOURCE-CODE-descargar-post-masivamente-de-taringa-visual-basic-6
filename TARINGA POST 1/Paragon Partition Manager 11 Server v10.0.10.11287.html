Componentes del producto

Con el fin de hacer frente a diferentes tareas, el producto contiene varios componentes:

conjunto de servicios basados en Windows es la parte fundamental del producto. Con la ayuda de un fácil utilizar el lanzador puede encontrar y ejecutar tareas de cierta complejidad en el campo de los datos y la protección del sistema, partición de disco duro y la clonación, etc

Linux / DOS entorno de recuperación basado en una multi-plataforma de medios de inicio que permite ejecutar utilidades en Linux o DOS PTS, y de esa manera tener acceso a su disco duro para fines de conservación o recuperación. El Linux / DOS entorno de recuperación no requiere instalación y puede ser de gran ayuda cuando el sistema no arranca. Además ofrece un Windows XP como el medio ambiente.
