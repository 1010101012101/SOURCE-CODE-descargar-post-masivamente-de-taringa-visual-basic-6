FUNCIONALIDADES
Edición de vídeo
El software Vegas Movie Studio HD Platinum 10 utiliza sencillos métodos de edición para arrastrar y soltar, así como comandos básicos, como cortar, copiar y pegar, para crear películas de aspecto profesional en su equipo. Puede personalizar sus películas con animaciones de texto, títulos y créditos de desplazamiento. Reproduzca vídeos a cámara lenta o rápida y hacia atrás o aplique efectos avanzados como una pantalla verde (croma) o una imagen dentro de otra.
Herramientas de audio
Puede hacer que la calidad del audio de sus películas esté a la altura de la calidad de vídeo gracias a potentes herramientas de audio. Éstos son algunos ejemplos de lo que puede hacer con el audio por medio del software Vegas Movie Studio HD Platinum: crear bandas sonoras personalizadas de música, eliminar el sonido no deseado de sus vídeos, utilizar envolventes para controlar el volumen y la panoramización del audio, grabar y editar audio en directo o sincronizar audio y vídeo.
Creación de DVD y Blu-ray Disc™
El software Vegas Movie Studio HD Platinum proporciona una amplia variedad de herramientas de creación de discos. Seleccione entre docenas de plantillas preparadas para ser utilizadas o cree sus propios fondos y botones personalizados. Cree títulos, varios menús, menús de selección de escenas, presentaciones de diapositivas, compilaciones musicales y mucho más. A continuación, grábelos en un DVD de definición estándar o un Blu-ray Disc de alta definición.

REQUISITOS
* Microsoft® Windows® XP (SP 3 o posterior), Windows Vista® (SP 2 o posterior) o Windows 7
* Procesador a 1 GHz (se recomienda un procesador múltiple o con varios núcleos para HD)
* 200 MB de espacio libre en disco para la instalación del programa
* 1 GB de memoria RAM (se recomiendan 2 GB para HD)
* Tarjeta de sonido compatible con Windows
* Conector i.LINK®/tarjeta IEEE-1394DV compatible con OHCI (para captura de DV e impresión en cinta)
* Conexión USB 2.0 (para importación desde cámaras de vídeo de AVCHD™ o DVD)
* Apple® QuickTime® 7.1.6 o posterior
* Conexión a Internet (para el servicio Gracenote MusicID)
* Unidad de DVD-ROM (sólo para la instalación desde DVD)
* Unidad de grabación de CD compatible (sólo para la grabación de CD)
* Unidad DVD-R/-RW/+R/+RW compatible (sólo para la grabación de DVD)
* Unidad de BD-R/-RE compatible (sólo para la grabación en Blu-ray Disc™)
* Microsoft .NET Framework 3.0 (incluido en el disco de la aplicación)
