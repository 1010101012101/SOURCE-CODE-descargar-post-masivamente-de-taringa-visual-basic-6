ActiveDen - Verve - Facebook Fan Page Template



DEMO:http://www.facebook.com/pages/Verve/161155993944211?sk=app_156972221033663

Verve is a Flash Template build for Facebook Fan Page Tab. It is XML driven and it has all the modules that are needed for a premium business solution, such as:

- presentation module that contains news and slideshow with text description
- text module with or without images
- gallery module with Ken Burns effect and Youtube support
- custom SWF module support
- external link
- contact module

Verve – Facebook Fan Page Template is highly reusable and highly customizable.

GENERAL FEATURES

* XML driven – NO FLASH KNOWLEDGE REQUIRED
* Ken Burns effect
* Youtube support
* Fully re-sizable; supports any height values and up to 520 width value
* HTML /CSS formatted text
* Create your own CSS properties
* Set individual time for each slide
* Custom Video Player with Youtube and normal video such as: MP4 , FLV, F4V , M4V, MOV
* Easily change the player skin by modifying the library assets
* Custom slide transitions effects: grid, fade, boxes, close, wave, horizontalStripes, verticalStripes, rectangles
* External link with target on the images: _self, _blank, _top
* Load separate background for each module
