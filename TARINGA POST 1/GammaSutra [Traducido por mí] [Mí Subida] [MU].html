Hola gente! Hoy les traigo este programa para aumentar y disminuir el gamma de nuestra PC facilmente. Que lo disfruten.


GammaSutra es una aplicacion para controlar el Gamma de nuestra PC facilmente usando un icono en la bandeja del sistema. El uso es muy sencillo, es como subir y bajar el volumen, solamente tenemos que desplazar la barra dependiendo el brillo que queramos, para arriba si queremos mas brillo y para abajo si queremos menos brillo.

Yo no creé este programa, solamente lo traduje al español y le hice un instalador para mas comodidad. La verdad que es un programa bastante util y a varios de ustedes les puede servir mucho. Lo recomiendo.
