Windows XP SP2 64 Bits


Si cuentas con un procesador de 64 bits puedes instalar esta versión de XP de 64 bits.
Obtendrás la misma interfaz, características y funciones, pero con un sistema operativo que aprovecha el acceso a toda la memoria.

La versión de 64 bits de Windows XP no es compatible con todos los procesadores.
Confirma que tu sistema, aplicaciones y dispositivos son compatibles con la edición de 64 bits de Windows XP antes de instalarla.

Requisitos del sistema Computadora de sobremesa o portátil con un procesador compatible:

* AMD Athlon 64, AMD Opteron, Intel Xeon con compatibilidad Intel EM64T, Intel Pentium 4 con compatibilidad Intel EM64T, Dual Core, Quad Core y superior
* 256 MB de RAM o superior
* 5 GB de espacio disponible en disco duro
* Unidad de CD-ROM o DVD
* Super VGA (800 x 600) o adaptador de vídeo de mayor resolución y monitor
* Teclado, mouse o dispositivo señalizador compatible
