Chainfire3D es un driver de OpenGL intermediario. ¿Qué significa eso? Esto significa que Chainfire3D se encuentra entre sus aplicaciones y los controladores de gráficos, y se puede interceptar y / o comandos de cambio entre las dos. Tiene algunas funciones incorporadas, y puede ser prorrogado nuevamente con los plugins para proporcionar funcionalidad adicional.

! Chainfire3D es de 1GHz + dispositivos sólo!

! RAÍZ se requiere para este programa, así como una escritura / partición del sistema. S-off también puede ser necesaria en los dispositivos de HTC!

Las características incluyen la reducción de tamaño de la textura y la calidad para reducir el uso de memoria y, potencialmente, incrementar el rendimiento del juego, así como aumentar la memoria de uso potencialmente aumentar el rendimiento.
