Detalles.
Windows 7 Total v2, es un todo en uno que incluye todas las ediciones de Windows 7 a excepcion de las versiones para servidores (Windows Server 2008 R2 x86/x64).
Todo los sistema incluido en el son originales, una de las principales caracteristicas es la autoactivacion de los sistemas al efectuarse la instalacion por defecto, incluso para la edicion enterpise. Tambien incluye un sistema de activacion diferente al ya conocido y herramientas de recuperacion (WinPE).
