MICROSOFT FRONTPAGE 2003 (IS0)
FrontPage 2003 proporciona las características, flexibilidad y funcionalidad para ayudar a crear mejores sitios Web. Incluye el diseño profesional, creación, datos y herramientas de publicación necesarias para crear sitios Web dinámicos y sofisticados.

Algunas de las características que forman parte de la última versión de FrontPage incluyen:

• Ayude a navegar a través de su sitio, y ver la estructura de archivos y su contenido, visual
• Las características incorporadas para HTML, CSS y JavaScript (parcial)
• Se incluyen editor de imágenes (Microsoft Image Composer)
• Punto y haga clic en la funcionalidad de las herramientas comunes, como mouseovers, formularios de correo electrónico, y cuenta con éxito
• Fácil de usar con conocimientos previos de los productos de Office
• Los datos de visualización integrada con los productos de Office como Access y Excel
• Soporte para temas basados ??en CSS (como las páginas maestras ASP.NET)
• Cuando se cambia la dirección URL de una página, todos los enlaces a esa página se cambia dinámicamente
• La tarea a la cesión de los proyectos de equipo
• El contenido se puede modificar desde cualquier lugar con FrontPage (la contraseña es necesario)
• Soporte para portapapeles ricos de importación de datos (es decir, copiar y pegar datos de Internet Explorer en FrontPage 2003 descargará automáticamente los recursos multimedia, como imágenes y guardarlos localmente)
• Soporte integrado para la web automatizada de plantillas, que incluyen automáticamente generados por el sistema multi-nivel de navegación.
• FrontPage 2003 introdujo el soporte para la web Dreamweaver compatibles plantillas, apodado "plantillas Web dinámicas".
