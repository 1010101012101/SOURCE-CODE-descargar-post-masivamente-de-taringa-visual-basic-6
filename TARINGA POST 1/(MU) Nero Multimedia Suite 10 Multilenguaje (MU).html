Nero Multimedia Suite 10 reúne en una solución única y eficaz tres productos que le aportarán posibilidades de trabajo insospechadas con los contenidos multimedia. Dispone de una amplísima gama de funciones avanzadas de edición de video, de herramientas de organización y reproducción de discos; de software de grabación de calidad superior, de tecnología para la creación de copias de seguridad confiables y de herramientas multimedia especiales con las que mejorará sus fotos, videos y música.

Requisitos del Sistema
* Windows® XP SP3 (32 bits), Windows Vista® SP1 (32 y 64 bits), Windows® 7 Home Premium, Professional o Ultimate (32 y 64 bits).
* Procesador AMD o Intel® de 2 GHz
* 512 MB de RAM (1 GB de RAM para Windows Vista® o Windows® 7)
* 5 GB de espacio en el disco duro para una instalación típica de todos los componentes (incluidas las plantillas, el contenido y el espacio en disco ocupado temporalmente).
* Se requiere una tarjeta gráfica compatible con DirectX® 9.0 con aceleración 3D, color de 16 bits y un mínimo de 128 MB de memoria de video (se recomienda disponer de 32 bits, color verdadero, memoria RAM de video dedicada de 256 MB y compatibilidad con Shader Model 3.0)
* Unidad de DVD para la instalación y la reproducción
* Unidad de CD, DVD o disco Blu-ray grabable o regrabable para la grabación y la creación de videos
* DirectX® 9.0c, revisión 30 (agosto de 2006)
* Windows Media® Player 9
* Los componentes de terceros (como Microsoft Windows® Installer 4.5, Microsoft .NET® 3.5 SP1 y Microsoft Silverlight™ 3) se suministran con el producto o mediante descarga automática en el caso de que no se incluyan en el paquete.
* Se precisa conexión a Internet para determinados servicios (se recomienda una conexión de DSL-1000 o superior). Los gastos de conexión a Internet son responsabilidad del usuario.


Nota:

* En los sistemas operativos de 64 bits, las aplicaciones se ejecutan en el modo de emulación de 32 bits.
* Nero DiscCopy Gadget no funciona en sistemas operativos de 64 bits.
* Para que el producto funcione correctamente, es preciso que los dispositivos estén correctamente instalados y el sistema operativo los haya reconocido.
* Se recomienda instalar los controladores de dispositivos certificados por WHQL más recientes.

Requisitos especiales / mínimos para la edición y creación de videos

* Controlador FireWire® compatible con OHCI (IEEE® 1394) para captura de DV o HDV™
* Tarjeta sintonizadora de TV o de captura de video para grabaciones de video digital o analógico
* Hasta 50 GB de espacio en disco duro para imágenes de disco y archivos temporales

Requisitos especiales / mínimos para funcionamiento con contenido de video de alta definición

* Procesador dual AMD o Intel® de al menos 2 GHz
* 2 GB de RAM
* Debe instalarse una tarjeta gráfica compatible con NVIDIA® CUDA™ para que se admita la tecnología AVC NVIDIA® CUDA™. Para obtener más información, visite: http://www.nvidia.com/object/cuda_learn_products.html
* Para la reproducción de contenido de video protegido contra copia, se requiere una configuración de hardware/software adecuada.

Requisitos especiales / mínimos para copia de seguridad

* Se requiere un disco duro formateado en NTFS para utilizar determinadas funciones de copia de seguridad y restauración.
* Se requiere acceso a Internet de banda ancha para las copias de seguridad en Nero Online Backup.

Requisitos especiales / mínimos para LightScribe y Labelflash™

* Grabador y soporte compatible con LightScribe
* Software host de LightScribe instalado
* Grabador y soporte compatible con Labelflash™
