Descripción de Ubuntu 11.04 Natty Narwhal:

Ubuntu es la distribución de Linux más popular del mundo dado su sencillez de uso, su constante crecimiento y todas las opciones y aplicaciones que incluye de forma nativa.

Ubuntu 11.04 Natty Narwhal está pensada para que todo el mundo pueda acceder a software de calidad, gratuito y de código abierto, lo que supone que siempre se encuentra evolucionando. Esto ha dado pie a que se forme una gran comunidad de usuarios que ofrece un apoyo constante hacia este proyecto.

Junto con Ubuntu, encontraremos una suite de programas que prácticamente cubren todo el espectro de necesidades que puedan surgir nada más instalar esta distribución. Entre esta colección podremos encontrar el navegador de la fundación Mozilla (Firefox), la suite ofimática de código abierto LibreOffice, el versátil editor de imágenes F-Spot, además de un gestor de correo, varios juegos y un sinfín de utilidades más para que podamos empezar a sacarle provecho a este genial sistema operativo desde el primer momento.
