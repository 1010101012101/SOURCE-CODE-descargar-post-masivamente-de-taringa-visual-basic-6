ESTUDIO DE REDES WIRELES

ERW 2.9.1.3
Este es el soft, basicamente es un lanzador de aplicacion que en este caso la principal será la suite de aircrack para windows, este soft esta diseñado al contrario de linux en botones que hacen de lo tedioso de escribir un montón de comandos en linux en simplemente dar un click.

El cometido principal de esta GUI es el estudio de nuestras redes y poder comprobar el grado de seguridad de las encriptaciones actuales, WEP, WPA...

En el pack tenemos todo lo necesario más el driver para las Atheros, si esta no es la tuya, descarga el soft commview foro wifi de Tamos.com y lo instalas, el te dirá si tu tarjeta es compatible y te instalará el driver, si quieres instalar un driver por tu cuenta, sigue los pasos de este manual.

Ahora veremos el ERW 2.8.1

MODO MONITOR
Para poner nuestra tarjeta en este modo, para ello es necesario que tengamos instalado el driver y que no es corriendo el commview ni otro programa de este tipo.

ETHERCHANGE
Esta opción no permite cambiar la MAC que es el numero de identificacion del aparato, esto es necesario si queremos usar otra mac en la falsa autentificació.

HEX/ASC
Utilidad para pasar de hexdecimal a códico asc “letras”

ASC/HEX
Lo contrario de HEX/ASC

CALCULATE

NOTEPAD
El bloc de notas de Windows XP

CPU
Administrador de tareas de XP, nos permite ver el consumo de nuestra CPU

CARD CHECK
Utilidad de tamos.com para ver si nuestra tarjeta es compatible con los drivers monitor.

TODAS
Nos mostrará todas las redes de todos los canales.

CANAL
Nos mostrará todas las redes de un canal, el canal será el que este en el campo canal de la configuración.

CMD
Utitlidad para ejecutar comandos en una ventana permantente.

FILES
Muestras todos los ficheros generados por los analisis de una configuración.

RESET
Elimina todos los ARPs y CAP de los analisis de una configuración.

FILES
Muestra todas las configuracion.

GRABAR
Guarda una configuración.

DATOS DE UNA CONFIGURACION

Nombre de la configuracion: lampi.ERW

FILE lampi.cap

Nombre del AP: Nombre de la red “ESSID”

Mac del AP: Lo comentado en etherchange. “BSSID”

Canal: canal donde se encuentra el AP

Mac cliente: cliente que esta asociado al AP

Mac falsa: Si queremos cambiarla por otra.

Security pass: key de encriptación de la red.

Recuadros en rojo

Puedes usarlo como quieras, yo los diseñe para guardar el admin y el pass de un router.

CAPTURA DE PAQUETES
Comienza la captura de datas de una red, la configuración actual.

FALSA.AUTENT.
Realiza una falsa autentificación a un AP con nuestra MAC. “x” cada x segundos “0” una vez.

INYECCION
Inyección de paquetes “x” cantidad pps

REINYECCION
Reinyecta un ARP generado por una inyección, esto para los equipos con CPU de 1core, hace menos uso de la CPU.

MODO AUTOMATICO
Realiza todo el proceso de arriba menos la reinyección.

FRAGMENTACION
Ataque A-5, intenta sacar un PRGA valido para convertirlo en un XOR.

XOR
Convierte el XOR generado por la fragmentación en un CAP preparado para la reinyección.

AIRCRACK-NG
Crackeador de contraseñas.

AIRCRACK 2.3
Antiguo crackeador de la suite, para cuando tengamos una cantidad muy alta de paquetes y el aircrack-ng no nos de la key.

AIRCRACK-NG-WPA
Crackeador de contraseñas para WPA, funciona con diccionarios.

D
Para la elección de un diccionario que utilizará el aircrack-ng para una WPA.

REDES ORANGE
Para redes de Orange tipo essid “adslxxxx”, requiere el uso de un diccionario. “En pruebas”

D
Genera el diccionario para el essid “adslxxxx” para las redes de Orange.

REDES WLAN
Para redes de imagenio, con muy pocos paquetes podemos dar con la key.
