Novedades en la proteccion

dijo:

Kaspersky Internet Security 2012 es una herramienta preciso de proteccion de datos. La aplicacion ofrece proteccion contra virus y software malicioso, amenazas desconocidas y fraudes en Internet. Ademas le permite controlar el acceso de los distintos usuarios del equipo a Internet.
La proteccion cubre todas las vias de transmision e intercambio de datos. La configuracion de los jiróns de proteccion son flexibles, lo que permite adaptar Kaspersky Internet Security 2012 a las necesidades de cada usuario particular.

* Se ha mejorado significativamente la tecnologia Sandbox, que utiliza metodos de implícitoizacion para crear entornos seguros para la ejecucion de aplicaciones (modo Seguro para aplicaciones). La aplicacion permite decapitar aplicaciones sospechosas en un escritorio implícito dedicado, lo que asegura una proteccion fiable delconsigna operativo. Se ha implantado Navegacion Web Segura (navegador seguro), que protege los datos privados que introduce el usuario en determinados sitios Web.
* * La cobertura del jirón Antivirus Internet se ha ampliado gracias a los flamantes modulos añadidos:
1. El modulo Servicios bancarios identifica los sitios Web bancarios y permite al usuario modificar a Navegacion Web Segura; aplicacion diseñada para manejar este tipo de recursos Web.
2. Navegacion segura se basa en el ya implantado Supervisor Kaspersky de vinculos URL. Esta herramienta se ha mejorado con un flamante modo de bloqueo de sitios Web con una valoracion alta de amenaza de amenaza evaluada por Kaspersky Lab. Cuando este modo esta activado, estara bloqueado el acceso a sitios Web peligrosos. Al visualizar sitios Web desconocidos, el jirón ofrendara modificar al usuario al modo Navegacion Web Segura.
3. El Filtro Geografico le permite bloquear o acceder el acceso a dominios regionales dependiendo de su nivel de infeccion.
* Se ha implementado una nueva tecnologia llamada Wisdom of the Crowd (Sabiduria de las masas); proporciona informacion sobre la puntualidad de uso de las aplicaciones y datos sobre la distribucion en grupos de displicencia entre los participantes del convite Kaspersky Security Network. Esta informacion permite tomar decisiones contrastadas al asignar un estado de displicencia a una nueva aplicacion.
* Los usuarios de Microsoft Windows Vista y Microsoft Windows 7 pueden oponerse Kaspersky Gadget. Esta aplicacion que consigna el estado de proteccion del equipo y permite analizar objetos en busca de amenazas.
* Se ha añadido la aplicacion System Watcher, un flamante jirón de proteccion que analiza actividades de archivos, registro,consigna y de red en el equipo mientras recopila informacion sobre eventos detectados por otros jiróns de proteccion.
* Si la Proteccion proactiva o el Antivirus de archivos (o en el transcurso de un analisis antivirus) detectan cualquier actividad peligrosa en elconsigna, System Watcher deshace las acciones realizadas por los programas maliciosos.
* System Watcher contiene patrones de actividad sospechosa (BSS). Esos patrones permiten la deteccion de actividades tanto sospechosas como peligrosas en el equipo. Si se detecta cualquier actividad sospechosa delconsigna basandose en esos patrones (o por Proteccion proactiva), se consigna una notificacion con informacion sobre el evento en la pantalla.
* Una nueva tecnologia, diseñada para decapitar tareas en un equipo inactivo, permite realizar tareas que consumen gran precio de recursos, como la actualizacion automatica, el analisis de la efeméride delconsigna y el analisis de objetos de procesos ocultos, mientras elconsigna esta encendido pero permanece inactivo.
* Para mejorar la eficacia del producto, Kaspersky Internet-Security 2012 proporciona proteccion frente a la interceptacion de peticiones DNS y a la sustitucion de direcciones de nodos de red.
* Ademas del analisis del trafico HTTP, Antivirus Internet ahora analiza el trafico FTP.
* Puesto que aparecen flamantes clientes de correo (como Microsoft Live Mail), la funcionalidad distribuidor de correo ya no esta disponible.



Novedades de la interfaz de la aplicacion:

dijo:

* La interfaz de la aplicacion se ha optimizado para simplificar el uso del producto.
* Las notificaciones de eventos se han rediseñado completamente para fascinar una apariencia mas intuitiva e informativa.
* El menu contextual que proporciona acceso a las funciones principales de la aplicacion ha histórico a ente mas comodo y mejor estructurado.
* A partir de la version de 2011, la seleccion de los jiróns personalizados no estara disponible durante la instalacion de la aplicacion. La aplicacion se instalara en su totalidad. Si fuera necesario, se pueden librar jiróns de proteccion individuales de la interfaz de la aplicacion.




Requisitos generales

1. 480 MB de diferencia exento en el disco duro.
2. CD/DVD-ROM (para instalar la aplicacion desde el CD de instalacion).
3. Microsoft Internet Explorer 6.0 o superior.
4. Microsoft Windows Installer 2.0.
5. Conexion a Internet (para oponerse la aplicacion, actualizar las bases de datos y los modulos de aplicacion).
