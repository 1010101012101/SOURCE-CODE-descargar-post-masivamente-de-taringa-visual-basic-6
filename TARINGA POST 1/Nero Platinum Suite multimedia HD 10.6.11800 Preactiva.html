Requisitos mínimos del sistema para Nero Multimedia Suite Platinum HD 10

SP3 de Windows XP (32 bits), Windows Vista Service Pack 1 o posterior (32 y 64 bits), Windows 7 Home Premium, Professional o Ultimate (32 y 64 bits)
2 GHz AMD o Intel, 512 MB de RAM (1 GB de RAM para Windows Vista o Windows 7)
Para reproducción de Blu-ray Disc y HD de edición de vídeo: Procesador Intel Core Duo II 2,2 GHz, AMD Athlon 64 X2 5200 + de 2,6 GHz o superior, 2 GB de RAM
5 GB de espacio en disco duro para una instalación típica de todos los componentes (incluidas las plantillas, el contenido y el espacio temporal en disco)
DirectX 9.0 compatible con tarjeta gráfica con aceleración 3D de 16 bits de color, y un mínimo de 128 MB de memoria de vídeo es necesario (32 bits, color verdadero, 256 MB de RAM de vídeo dedicada y soporte para Shader Model 3.0)
Para el formato Blu ray Disc: Tarjeta gráfica NVIDIA 8600 y anteriores, ATI Radeon HD 3800 y superiores, Intel G45 (Windows Vista / Windows 7); Mínimo de 512 MB de VRAM, pantalla compatible con HDCP para la conexión digital
DVD unidad de disco para la instalación y la reproducción, Blu ray Disc capaces de unidad para la reproducción de BD
rayos de CD, DVD o Blu Disc grabable o regrabable para grabaciones de vídeo y autoría
DirectX 9.0c revisión 30 (agosto de 2006)
Windows Media Player 9
componentes de terceros, como Microsoft Windows Installer 4.5 y Microsoft. NET 4 se envían con el producto o descarga automática si no está incluido en el paquete
Para algunos servicios y el uso de BD Live cuenta con una conexión a Internet es necesario (DSL 1000 y superior). gastos de conexión a Internet son responsabilidad de los usuarios
Para conocer los requisitos, póngase en www.nero.com de verificación

Tenga en cuenta:

En sistemas de 64 bits operativo, las aplicaciones se ejecutan en modo de emulación de 32 bits
Nero DiscCopy Gadget no se ejecuta en sistemas operativos de 64 bits
Los dispositivos deben estar correctamente instalados y reconocidos por el sistema operativo para que el producto funcione correctamente
La instalación de los controladores más recientes dispositivos con certificado WHQL es muy recomendable

Requisitos especiales / mínimos para la edición de vídeo y autoría

Controlador FireWire OHCI compatible (IEEE 1394) para la captura de DV o HDV
Sintonizador de TV o tarjeta de captura de vídeo para la grabación de vídeo analógico y digital
Hasta 50 GB de espacio en disco duro para imágenes de disco y los archivos temporales
Para el apoyo de la tecnología NVIDIA CUDA una tarjeta gráfica compatible debe estar instalado y un mínimo de 256 MB de memoria de vídeo es necesario.

Requisitos especiales / mínimos para copia de seguridad

Un disco duro formateado en NTFS es necesario para el uso de copia de seguridad de algunos / restauración de instalaciones
Acceso de banda ancha a Internet es necesario para copias de seguridad de Nero Online Backup

Requisitos especiales / mínimos para LightScribe y Labelflash

Grabadora compatible con LightScribe y medianas empresas.
software de LightScribe debe ser instalado.
Labelflash compatibles grabador y medianas empresas.
