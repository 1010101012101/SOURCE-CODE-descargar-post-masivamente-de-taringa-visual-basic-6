Tu Esclavo 1.2

Descripción:

Es un sistema de control por de texto y/o voz para activar acciones en el ordenador a través de un "esclavo", que no es más que uno de los agentes de Microsoft.

Todas las acciones se basan en la realización de las mismas por parte del esclavo (un genio, un mago, un loro...), y para conseguir que las lleve a cabo se requerirá como mínimo una de las tres opciones siguientes: de manera oral, por escrito, y automatizado por acción/reacción. Podremos hacer casi de todo, como por ejemplo ejecutar aplicaciones, chequear cuentas de correo, avisar de archivos descargados, leer texto, etc. La configuración es bastante completa, incluyendo agendas y acciones recopiladas en una base de datos Access muy simple.

Tal como su nombre indica, Esclavo está diseñado para obedecerte ciegamente a través del control de tu voz, y para avisarte de las tareas que tienes pendientes.

Aquí te ofrecemos algunos ejemplos de sus comandos generales, siempre en español:

Vete Esclavo: Sale del Programa diciendo antes el mensaje de Despedida.

Saluda esclavo: Saluda al Usuario.

Dime la hora esclavo: Te dice la Hora actual.

Escóndete esclavo: Oculta el Agente.

Muéstrate esclavo: Muestra el agente .

Lee el portapapeles esclavo: Lee el Contenido del Portapapeles, si hay texto.

Dime mi IP esclavo: Te dice tu IP y la copia al Portapapeles.

Comprueba el correo esclavo: Comprueba el Programa de Correo Predeterminado (normalmente Outlook Express).

Apaga el ordenador esclavo: Apaga el ordenador.

Reinicia el ordenador esclavo: Reinicia el Ordenador.

Cállate esclavo: Para lo que este diciendo o haciendo el agente al momento.

Hazte el sordo esclavo: Hace que el esclavo no haga caso a lo que oiga, sean comandos o frases.

Escucha esclavo: Hace lo contrario que el comando anterior.

Requiere un motor de habla TTS para leer texto y un motor de reconocimiento de voz SR para obedecer órdenes habladas, aunque ninguno de los dos son indispensables.
