Configurar jDowloader como servicio de Windows

Para ello debemos descargar he instalar de la pagina oficial de Microsoft esta aplicación: Windows Resource Kits, el archivo de instalación lleva por nombre: rktools (y pesa 11 mb aprox.)
Una vez instalado debemos agregar un servicio con el comando “instsrv” desde el cmd de Windows para tal fin van a ir a Inicio/ejecutar…si no encuentran el ejecutar pueden iniciarlo con Win+R o clic derecho en la barra de tareas, clic en administrador de tareas y luego en clic en tarea nueva y escribir cmd y enter.
Escribir: (a modo de ejemplo) (para Windows xp) instsrv "Nombre de nuestro servicio" "C:Archivos de programasWindows Resource KitsToolssrvany.exe"
Por ejemplo en nombre de programa yo le puse jd3: Ver imagen con total atención ver todo.
Luego del mensaje:

The service was successfuly added!

Make sure that you go into the Control Panel and use
the Services applet to change the Account Name and
Password that this newly installed service will use
for its Security Context.

Cerrar el cmd y ejcutar nueva tarea, escribir Regedit y enter. Y hacer lo siguiente:
buscamos la ruta:
HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesNombre de nuestro servicio

Hacemos clic derecho sobre la carpeta y le hacemos clic en Nueva>Clave
Creamos la Clave con el nombre de Parameters. Dentro de esta clave debemos crear un nuevo “Valor alfa numérico” con el nombre de "Application". Dentro de este debemos colocar la ruta de inicio de la aplicación.
