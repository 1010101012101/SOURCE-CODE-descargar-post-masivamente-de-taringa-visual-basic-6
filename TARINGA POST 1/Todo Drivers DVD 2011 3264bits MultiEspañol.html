Descripcion

Este DVD contiene todos los controladores de hardware y accesorios para cualquier equipo de sobremesa o portátil también es compatible con todos los sistemas de ventanas (x86/x64): XP, Vista, Win7, Server 2003 y Server 2008.
Es fácil de usar con aspecto simple para instalar el controlador directamente. Dispone de un sistema inteligente para detectar y analizar todos los dispositivos conectados y el hardware en su computadora también la capacidad de ir a buscar cualquier controlador de las marcas famosas como Dell, HP, Acer, Compaq, IBM, Sony, Toshiba, Panasonic, VIA, nVidia, ATI, SoundMax y muchos más ..

Dispositivos

* Chipset
* Video
* Audio
* NIC y los módems PCI
* Controladores de disco duro
* IEEE 1394 Host Controller
* Los controladores USB Host
