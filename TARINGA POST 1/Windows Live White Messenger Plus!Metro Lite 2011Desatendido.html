Windows Live Messenger cuenta con una interfaz renovada. Inmediatamente después de su lanzamiento, y gracias a la nueva integración con otros servicios como Facebook, MySpace y YouTube, el más destacado sociales llamada le permite ver sus amigos las últimas noticias y mensajes de estado de un vistazo. Si aún prefiere el viejo estilo, no se preocupe: Windows Live Messenger te permite cambiar a la lista de contactos tradicionales utilizando el modo de Contactos. Tenga en mente que si usted utiliza una cuenta de correo electrónico diferentes para sus redes sociales, tendrá que conectar los servicios del sitio web de Windows Live.

Además del elemento social, todo lo demás es casi lo mismo en Windows Live Messenger 2011, sólo con un aspecto ligeramente más fresco. Usted todavía puede hacer llamadas de video y voz, jugar juegos en línea con tus amigos, compartir archivos, y por supuesto, personalizar la apariencia del programa con las pieles y los temas de color. Emoticonos se han renovado, y – ¡por fin! – usted puede tener múltiples conversaciones con pestañas en una sola ventana.
El principal inconveniente para el nuevo Windows Live Messenger 2011 es que sólo funciona en Windows Vista y Windows 7. Así que si usted todavía utiliza Windows XP, tendrá que atenerse a la versión anterior – o actualizar su sistema!
