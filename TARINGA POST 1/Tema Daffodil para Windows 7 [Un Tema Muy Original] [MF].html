Tema Daffodil para Windows 7 [Un Tema Muy Original] [MF]


Taringueros!,Navegando encontre un tema para Windows 7 que me llamo bastante la atencion, pues se veia bastante bien, pero dude ya que muchas veces no cumplen lo que prometen, afortunadamente este no es el caso. Daffodil es un tema que mejora por mucho la apariencia de Windows, dejandolo mas elegante y a la vez moderno, sin mas les dejo unas imagenes.







4.99Mb | MF | Compatible con Windows 7 Con o Sin SP1 |

Descargár


Antes que nada debes tener instalado el Universal Theme Patcher:

1.Descomprime el rar en una carpeta donde la puedas ubicar facilmente

2. Copia el contenido de la carpeta “Temas” en esta ruta: “C:WindowsResources”

3. Descarga esta herramienta para tomar el control de un archivo: http://www.mediafire.com/?gl1qogu6zgcva4f, despues de bajar y descomprimir el rar, te quedara una carpeta con dos archivos, abre el que se llama “Take Control”, en la ventana que sale selecciona el boton “+Add”, se abrira otra ventana y tendras que ir a esta ruta “C:WindowsSystem32? ya en esta carpeta busca el archivo llamada “ExplorerFrame.dll”, dale doble clic y ahora selecciona el boton “Take Control”. Cierra el programa y regresa a la carpeta donde estaban los archivos que bajaste al principio.

4. Entra a la carpeta “System files” y luego abre la carpeta que te corresponda segun tengas un sistema de 32x o 64x, copia el archivo que encontraras dentro llamado “ExplorerFrame.dll”, ahora ve a esta ruta “C:WindowsSystem32? y busca el archivo llamado “ExplorerFrame.dll” y cambiale el nombre por “ExplorerFrame2.dll” y por ultimo pega el archivo que habias copiado.

5. En cualquier parte vacia del escritorio da clic derecho y selecciona “Personalizar”, en la ventana que te sale busca el tema que se llama Daffodil.xxx.xxx, para cual seleccionar lee lo siguiente

-Daffodil.top.visible: Selecciona este si tienes la barra de tareas arriba y quieres que se vean los botones de cerrar, maximixar y minimizar en todo momento

-Daffodil.top.hide: Selecciona este si tienes la barra de tareas arriba y quieres que los botones de cerrar, maximixar y minimizar solo se vean cuando pases el mouse por encima

-Daffodil.Bottom.Visible: Barra de herramientas abajo, los botones siempre se muestran

-Daffodil.Bottom.Hide: Barra de herramientas abajo, los botones solo se ven cuando pasas el mouse por encima

Despues de seleccionar el tema, reinicia tu equipo o cierra e inicia de nuevo tu sesion.

6. Notaras que faltan algunos detalles. Ve a la carpeta “Herramientas” y copia la carpeta “Hide Blur” y “Titlebar Icons Blanker” a un lugar donde no las vayas a mover ni borrar por ejemplo en “C:”

Abre la carpeta “Hide Blur” y abre el archivo “Patch xx-bit System Files” segun sea tu caso 32x o 64x y abrira otra ventana, solo presiona enter, es importante que corras el archivo como administrador y no este activado el Control de Cuentas de Usuario (UAC). Con esto se desaparecera la parte borrosa que quedaba. Para terminar en algunas ventanas como el Administrador de Tareas, hay un icono en la parte de arriba, si deseas quitarlo ve a la carpeta “Titlebar Icons Blanker” y abre el archivo “TBarIconBlanker” no notificara ni abrira nada, pero ya no se mostraran estos iconos. Listo!
