Windows Server 2008 R2 se diseñó sobre la premiada base de Windows Server 2008 para expandir la tecnología existente y agregar nuevas funciones que permitan a las organizaciones aumentar la confiabilidad y la flexibilidad de sus infraestructuras de servidor.

Este software está destinado a evaluación y pruebas. La evaluación está disponible en formato ISO. Las ediciones Web, Standard, Enterprise y Datacenter están disponibles mediante la misma descarga. Se le pedirá la instalación de la edición durante la configuración. No es necesario introducir una clave de producto para realizar la evaluación de cualquier versión del software Windows Server 2008 R2. Sin embargo, se exigirá la activación en un plazo de 10 días. Si no se activa la evaluación, el servicio de licencias apagará el equipo cada hora. (El período de activación de 10 días puede restablecerse cinco [5] veces con el comando rearmar. Consulte a continuación para obtener más información sobre el rearmado de activación). Una vez activada, la evaluación se ejecutará por 180 días. Cuando finalice la evaluación, deberá desinstalar el software o actualizarlo a una versión completa con licencia de Windows Server 2008 R2.
Cómo rearmar manualmente el período de gracia de activación de 10 días
Cuando el período inicial de activación de 10 días esté llegando a su fin, podrá ejecutar la secuencia de comandos Slmgr.vbs para que vuelva a comenzar. Para ello, siga estos pasos:
Haga clic en Inicio y, luego, en Símbolo del sistema.Introduzca slmgr.vbs -dli y presione ENTRAR para verificar el estado actual de su período de activación.Para restablecer el período de activación, introduzca slmgr.vbs –rearm y, luego, presione ENTRAR.Reinicie el equipo.Así se restablecerá el período de activación a 10 días. Este procedimiento se puede realizar hasta cinco veces. Después deberá activar o reinstalar la evaluación.
Estos procedimientos no son necesarios y usas el Loader.
