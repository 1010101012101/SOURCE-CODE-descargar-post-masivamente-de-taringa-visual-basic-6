EMDB es una aplicación que permite mantener organizada de forma correcta una colección de películas.

Al igual que en otros muchos programas de este tipo, EMDB no necesita que el usuario introduzca la información relativa a una película, basta con indicar el nombre del filme para que EMDB se encargue de recopilar los datos de Internet.

La fuente de información que emplea el programa es IMDB (Internet Movie DataBase o www.imdb.com).

Los campos de datos que puede almacenar EMDB son los siguientes: título, nombre alternativo, versión, director, país, año, duración, género, trama, nombre de los actores, formato, idiomas, resolución, fuente, etc. EMDB también recupera y guarda una miniatura de la carátula de lapelícula.

Además, y entre una multitud de opciones útiles propias de un programa de este tipo, podremos obtener una vista previa de las carátulas, la posibilidad de hacer un seguimiento de los préstamos efectuados y una flexible y completa función de búsqueda.
