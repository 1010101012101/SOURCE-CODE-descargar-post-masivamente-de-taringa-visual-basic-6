AnyToISO es una sencilla aplicación que permite a los usuarios convertir sus archivos en imágenes ISO. El mejor creador de la ISO para Windows: puede crear la ISO de casi todo, incluyendo todos los formatos de CD / DVD de las imágenes populares de Internet ( NRG, MDF, UIF, DMG, ISZ, BIN, DAA, PDI, CDI, IMG, etc ), CD / DVD / discos Blue- ray o simplemente de una carpeta local.


Características principales

- AnyToISO soporta todos los formatos de CD / DVD populares en Internet.
- La mayoría de CD / DVD software de grabación de las obras con las imágenes ISO y sus imágenes nativos. Utilice AnyToISO para la conversión antes de grabar.
- Con AnyToISO puede montar cualquier imagen de CD / DVD que se encuentran en Internet para su Parallels / máquina virtual de VMware.
* El programa es gratuito para convertir imágenes de CD, una licencia profesional permite convertir imágenes de DVD.
AnyToISO apoya los CD / DVD de formatos de imagen populares en Internet.
- convertir la imagen al formato de ISO
- extraer la imagen en una unidad local
