Simfatic Forms v3.2.1.252


Simfatic formas - utilidad que te permite crear rápidamente formularios para su sitio web. Usando el software, puede crear un "boceto" de la futura configuración y añadir una serie de elementos básicos y que los datos necesarios, entonces, sólo necesitan un "clic" del ratón para generar automáticamente el programa que necesitas código HTML. A su vez, el usuario no está de ninguna manera participar en el proceso de su escritura. La utilidad cuenta con el apoyo, ya que todos los elementos originales de las páginas electrónicas, y más complejo: una ventana para subir archivos al sitio, la forma de una confirmación automática de colocar información sobre el recurso (enviado en un orden automático a la dirección del material publicado autor). El proceso de agregar una forma similar a la estructura de lo creado páginas ya no son también difíciles y se realiza con un solo clic.



Formas Simfatic es una herramienta de software para crear formularios para tu sitio web. Todo lo que tienes que hacer es diseñar la forma en que quieres que tu forma de mirar, usando el editor visual. A continuación, puede elegir las opciones de procesamiento de formularios (como validaciones de entrada, correo electrónico y el ahorro de los envíos de formularios y respuesta automática.) Formas Simfatic genera el código y le ayuda a integrar estrechamente el formulario con su sitio web.

Las características principales de las formas Simfatic
Usar formularios Simfatic, puede crear rápidamente formas, instalarlo en su sitio web y comienza a recibir envíos de formularios.
Editor WYSIWYG flexible
Uso del WYSIWYG flexibles (lo que ves es lo que obtienes) editor de formularios puede diseñar la forma de la manera deseada.

Fácil, la interfaz paso a paso

El mago-como interfaz que permite crear la forma en pasos fáciles.
Configuración rápida

Estos son los elementos de entrada predefinidos que se pueden agregar al formulario con sólo seleccionar un elemento de menú. (Nombre, correo electrónico, dirección, etc) Usted puede agregar sus propios elementos prefabricados para la configuración rápida.
CAPTCHA para proteger la forma
Utiliza la función Captcha para proteger la forma de los robots de spam automatizado.
Obtener envíos de formularios instantáneamente por e-mail
Usted puede dirigir la envíos de formularios a cualquier número de direcciones de correo electrónico.
Personalizar el correo electrónico
Puede crear una plantilla de correo electrónico para el envío del formulario de correo electrónico. Esto le permite poner de relieve la parte de los datos del formulario de presentación o formato de los datos de envío del formulario en la forma que desee.
Guardar en la base de datos
Usted puede tener la envíos de formularios guardados en una tabla de base de datos. formas Simfatic creará automáticamente y actualizar la tabla según sea necesario.
La versión actual soporta base de datos MySQL.
Guardar los envíos de formularios a un archivo
Además de recibir los envíos de formularios por correo electrónico, usted puede tener la forma de presentación de datos guardados en un archivo csv (valores separados por comas) archivo en el servidor. Más tarde, puede descargar este archivo y abra la presentación de datos en los programas de hoja de cálculo o de importación a una base de datos en formato csv apoyo.
Enviar una respuesta automática
Puede configurar un correo electrónico de respuesta automática que se envía al visitante que presentó el formulario.
Personalice el correo electrónico mediante plantillas de correo electrónico
Usted puede personalizar el correo electrónico de respuesta automática para que sea más personalizada.
Mensajes de correo electrónico HTML
Usted puede tener formato de mensajes de correo electrónico HTML. (Con texto con formato, fuentes, etc)
Analizar la forma de una página web existente

Usted puede dejar que las formas Simfatic leer un formulario de una página Web existente y adjuntar las validaciones y el procesador de forma Simfatic a la forma.

Por lo tanto usted puede añadir soporte de procesamiento de formularios a los formularios existentes.

Del mismo modo, si usted desea crear un formulario en su editor favorito y adjuntar el apoyo de procesamiento de formularios a la misma, usted puede utilizar esta función.
Cree rápidamente formulario utilizando una plantilla
Puede crear rápidamente la forma de utilizar esta función. Seleccione una plantilla a partir de plantillas de uso común y crear un formulario utilizando la plantilla. Esto hace que la creación de formas aún más rápido.
Personaliza el aspecto de la forma
Es fácil de personalizar la apariencia de la forma mediante la selección de la fuente, el color y el color de fondo.

Hay pre-construidos "estilos" que puede adjuntar a la forma que está creando.
Rápida cargar: para que se integra fácilmente la forma
Usted no tiene que preocuparse de cargar los archivos de comandos individuales a la ubicación correcta en su servidor web. Los formularios pueden Simfatic FTP los archivos necesarios al servidor.
