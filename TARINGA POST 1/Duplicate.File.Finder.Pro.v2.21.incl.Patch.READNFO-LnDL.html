Duplicate File Finder






Duplicate File Finder Pro mantiene tu pc libre de archivos innecesarios encontrando y eliminando archivos duplicados de tu elección en tus discos.

Flexibles Opciones de Escaneo
Duplicate File Finder Pro te ofrece la habilidad de añadir filtros para cada ruta que quieras escanear. Puedes poner, por ejemplo, que solo busque archvios *.mp3 en C:Musica. Hay opciones de filtrado de archivos basadas en el tamaño,atributo,nombre,etc.

100% Preciso al comparar archivos
Cuando el primer escaneo inicial haya concluido, entonces el programa te permite combinar hasta tres algoritmos diferentes para comparar los archivos tan pronto como sea posible con un 100% de acierto. Esto significa que puedes estar seguro que los archivos encontrados por Duplicate File Finder son realmente archivos duplicados.

Selecciona los archivos de tu elección
Después de comparar los archivos, todos los duplicados son automáticamente seleccionados. Pero si tienes necesidades específicas sobre cúales archivos serán procesados, entonces puedes usar la función Seleccionar archivos para seleccionar archivos en la lista basado en el tamaño, nombre y ruta.

Elimina o mueve los duplicados
El último paso es seleccionar si quieres mover o eliminar los duplicados. Puedes enviarlos a la Papelera de Reciclaje,eliminarlos permanentemente, o moverlos a un directorio de tu elección.

Exporta reportes detallados
Una vez el escaneo ha terminado y los duplicados han sido encontrados, puedes exportar un reporte detallado de los archivos. El reporte puede ser grabado en HTML y TXT.
