Si tienen varias cuentas de correo en Gmail habrán comprobado que resulta imposible abrir varias sesiones distintas de forma simultánea. Aunque traten de hacerlo utilizando diferentes pestañas o ventanas, el servicio cerrará la sesión anterior.



Pero todo tiene un truco, y en este caso se trata de una extensión para el navegador Firefox. Sí que es posible visualizar diferentes cuentas de Gmail al mismo tiempo, y hoy les vamos a enseñar cómo pueden hacerlo.

En primer lugar deben ser usuarios o utilizar un complemento llamado Multifox. Esta extensión les permitirá abrir varias sesiones utilizando pestañas, y no solo es aplicable a las cuentas de Gmail, sino que también funciona para otros servicios como Facebook y Twitter.

Dicho esto, pueden dirigirse al sitio de Multifox para instalar el complemento en su navegador. Tras finalizar la instalación están preparados para acceder a varias cuentas de Gmail, Twitter o Facebook.

Ahora pueden dirigirse a cualquiera de estos servicios, por ejemplo Gmail, y hacer login con una de sus cuentas. Una vez logeados coloquen el puntero del mouse sobre cualquier enlace al propio sitio, por ejemplo el logo de Gmail, y hagan clic con el botón derecho del ratón para que se abra el menú contextual.

En este menú aparecerá una nueva opción con el texto “Open in a New Identify Profile“, como puede observarse en la imagen superior, sólo deben seleccionarla para que se abra una nueva pestaña donde ya podrán loguearse con otra cuenta diferente.
