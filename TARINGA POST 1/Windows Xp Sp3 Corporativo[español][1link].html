Descripción
Windows Semi-Desatendido con las ultimas actualizaciones integradas, ya se encuentra activado. Lleva cambiado el tema por defecto de windows, al iniciar windows por primera vez se instala . NET Framework.

Integraciones
* Se ha integrado Windows Media Player 11 (WMP11) como una actualización mayor.
* Se ha integrado Windows Internet Explorer 8 (IE8) como una actualización mayor.
* Se ha integrado Conexión de Excritorio Remoto (RDC7) como una actualización mayor.
* Se ha integrado Windows Messenger 5.1 (WM5.1) como una actualización mayor (opcional, eliminada de los servidores de microsoft).
* Se ha integrado Silverlight 4.x.
* Se ha integrado Java Update 21.
* Se ha integrado Shockwave Player.
* Se ha integrado Flash player.
* Se ha integrado .NET Framework 1.0.
* Se ha integrado .NET Framework 2.0.
* Se ha integrado .NET Framework 3.0.
* Se ha integrado .NET Framework 3.5.
* Se han integrado todas las actualizaciones críticas y recomendadas para windows hasta el 24 de Agosto de 2010.
* Se ha integrado un paquete de temas oficiales firmados por microsoft.
Eliminación
* Se eliminó Tour de windows (alguien necesita conocer lo “nuevo” de XP?).
* Se eliminó Cursores de Ratón (tan monos que muchos se instalan los de XP o 7, que es lo mismo).
* Se eliminó Música de Ejemplo (solo sirve para probar que sale el audio del equipo si no hay nada de audio o música y si no se sabe donde están los sonidos de windows).
* Se eliminó Asist. para Transferencia de Archivos y Configuraciones (alguien necesita transferir su configuración a otro equipo?).
* Se eliminó Asistente de Limpieza del Escritorio (una brocha y/o aire comprimio es mejor y muy molesto si aparece cuando algo muy importante o interesante está en proceso).
* Se eliminó Centro de Seguridad (que de seguridad, no tiene nada, es molesto cuando deshabilitas el antivirus o algúin otro motivo por el que tenga que saltar).
* Se eliminó DR Watson (sirve para eliminar errores pero es como los políticos, gasta recursos pero no da resultados).
* Se eliminó Framework (el cual está incluido en el CD pero no se instala, en su lugar integre el :NET Framework 1.0 SP3).
* Se eliminó Liberador de Espacio en Disco (algo molesto cuando queda el 10 % de espacio en el disco, creo que debí dejarlo).
* Se eliminó MSN Explorer (necesario tener cuenta MSN, en USA al contratar los servicios de Qwest Internet puede utilizarse, de lo que recuerdo, en MX no es posible).
* Se eliminó “Peer-to-Peer” (Supuestamente es un P2P microsot que solo sirve para redes microsoft, si reemplazara a emule, ares, pando o torrents otra cosa sería).
* Se eliminó Horario de Windows (necesario si se debe sincronizar la hora cada cierto tiempo, creo que es más fácil cambiar la hora manualmente si el equipo no tiene bateria para CMOS).
* Se eliminó Servicio de Indexación (puede servir para búsqueda rápida no tan rápida, lo veo como pérdida de recursos).
Cambios
* Se ha ocultado “Notificaciones de WGA” (No aparece para instalar “Windows Genuine Advantage”).
* Se ha ocultado Windws Search (no aparece para instalar en windows update).
* Se ha cambiado el máximo de la lista documentos recientes a 30.
* Se agregó la eliminación del mensaje al inicio de sesión “Tiene ## correos no leidos”.
* Se ocultó la instalación de la heramienta de software malintencionado (solo de agosto).
* Se ha cambiado la imagen de inicio de sesión, un poco más “xpized” con la leyenda “Windows XP Professional Edition”.
