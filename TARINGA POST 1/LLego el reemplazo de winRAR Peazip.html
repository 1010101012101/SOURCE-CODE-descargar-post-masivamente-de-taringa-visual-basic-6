Consejos

PeaZip es un archivador libre, disponible como software instalable y portable
para Windows y Linux.
Crear 7Z, ARC, BZ2, GZ, * PAQ, PEA, QUAD / Balza, TAR, UPX, ZIP
Extracto de ACE, ARJ, CAB, DMG, ISO, LHA, RAR, UDF y muchos formatos de archivo más
PeaZip permite crear, convertir y extraer múltiples archivos a la vez,
crear archivos auto-extraíbles, los archivos y carpetas de favoritos, se aplican
poderosos filtros de búsqueda, exploración y abrir archivos con aplicaciones personalizadas,
aplicar el cifrado fuerte división, / unir archivos, borrado seguro de datos,
cálculo de suma de comprobación y hash ...

En las redes, la aplicación puede ponerse a disposición de varios usuarios,
ya sea un poco con una configuración común o con la configuración por separado para
cada usuario, consulte el apartado "Configuración" y "Personalización y secuencias de comandos" en la ayuda
archivo, que también contienen consejos útiles para mejorar el sistema de PeaZip
integración.

Instalar / desinstalar en Windows
Para ejecutar la instalación desatendida en marcha el programa de instalación de Windows con / SILENT
parámetro, el uso / VERYSILENT para la instalación de fondo, sin
notificación a los usuarios.
Para volver a configurar la integración de sistemas (asociaciones de archivos, menú de contexto y
enviar a las entradas de menú), puede utilizar Opciones> Configuración> Sistema
integración de enlace.
Para usar la aplicación en secuencias de comandos, consulte la documentación
(F1).
En la instalación, PeaZip crea algunas entradas en el menú de SendTo útiles que pueden ser
simplemente se copia a un usuario a otro, según sea necesario, que se encuentran en
C: Documents and Settings (nombre de usuario) SendTo (Windows XP) o C: Users
(Nombre de usuario) AppData Roaming Microsoft Windows SendTo (Vista / 7), suponiendo que C
como disco de sistema.
Cuando desinstalado, restaura archivos PeaZip asociación con Windows "
Carpetas comprimidas, para los tipos de archivos compatibles.
En cualquier momento, para restaurar manualmente las asociaciones de archivos comprimidos con
utilidad carpetas ejecutar: cmd / c = Asociación postal CompressedFolder sustituir postal..
con cualquier extensión de archivo que se desea para ser asociado a Comprimido
Carpeta de servicios públicos.
Para restaurar las asociaciones de archivos con un software archivador anterior por favor
se refieren a su propia documentación, por lo general se puede hacer correr el
programa de paquete de instalación o desde la propia aplicación
