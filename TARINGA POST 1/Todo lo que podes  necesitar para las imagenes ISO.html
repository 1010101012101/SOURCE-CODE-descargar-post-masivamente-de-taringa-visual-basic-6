En esta "entrega", le dejo a toda la comunidad taringuera varias herramientas necesarias para la creación y montaje de las imagenes ISO.

Primero en principal les voy a explicar mas o menos que es una imagen ISO, una imagen ISO es un archivo que es utilizado para almacenar copias de sistemas de ficheros, todo esto por lo general se hace en un CD o DVD, aunque se pueden crear discos virtuales que contengan la imagen tal cual lo haría un CD. Para el que no entendió una imagen ISO sería una caja con una cadena adentro.

Si quieren saber mas sobre este tipo de documentos dale click a las barritas azules |||||
