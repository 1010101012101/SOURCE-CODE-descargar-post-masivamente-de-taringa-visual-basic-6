¿Qué hay de nuevo en Rainmeter 2.1 beta
[Traducido del ingles]

Agregó el jugador NowPlaying los medios de comunicación por el desarrollador de plug-in y poiru general superestrella Rainmeter.Este es un nuevo plugin para apoyar a los reproductores de medios múltiples, con soporte completo para AIMP, Foobar200, MediaMonkey, MusicBee, iTunes, WinAmp y WMP.  Apoyo parcial también se proporciona para Spotify.

Añadido soporte para Aero Glass ” desenfoque ” efectos sobre la piel!
Esto le permitirá tener el efecto Aero de Windows, en el fondo del escritorio es ” borrosa ” detrás de sus pieles o partes de pieles.  Windows Vista o Windows 7 se requiere o la nueva configuración se ignoran por Rainmeter.  Ver Aero Blur para más detalles y un ejemplo de la piel.

Mejoras en FolderInfo plugin.
FolderInfo ahora devuelve el tamaño en bytes.  Si se utiliza InfoType = FolderSizeStr, ( que hasta ahora autoscaled el valor interno ) AutoScale = 1 hay que añadir al contador ( s ).  Si bien esto no cambia el comportamiento, que permite el uso de la escala, NumOfDecimals, etc que no eran posibles antes de este cambio, sin una medida adicional Calc.

Arreglos InputText plugin no si un ” config nombre de ” carpeta había un espacio en ella.

Arreglo para el CoreTemp plugin para hacer frente a una exclusión mutua pérdida de identificadores.
