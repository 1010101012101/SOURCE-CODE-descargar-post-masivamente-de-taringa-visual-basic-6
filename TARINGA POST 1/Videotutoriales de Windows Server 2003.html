En este curso en videotutoriales, promovido por www.soportegallo.blogspot.com se desea proveer los conocimientos necesarios para poder Administrar Servidores con todas las aplicaciones y servicios que ofrece Microsoft Windows Server 2003. El Curso se desarrolla en tres partes: La primera, todo lo relativo a la Administración de Redes LAN, la Segunda se trata Active Directory como herramienta vital de Administración y la Tercera, Servicios Web. Casi todos los videos tienen una duración promedio de 50 minutos.

Todos los videotutoriales han sido desarrollados en consolas virtuales lo que significa que es un curso práctico, pero siempre en la parte introductoria se definen algunos elementos que son de gran importancia para la comprensión absoluta de las temáticas tratadas.
