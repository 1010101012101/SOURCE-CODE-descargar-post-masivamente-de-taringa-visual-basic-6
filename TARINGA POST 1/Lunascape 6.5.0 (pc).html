Lunascape es un navegador web gratuito para Windows que nos permite visualizar páginas web a través de diferentes motores de renderizado.

Lunascape nos permite cambiar entre varios motores de renderizado al visualizar una web. Entre los disponibles podemos encontrar a Trident (IE), Gecko (Firefox) y varias variables de WebKit (Safari, Chrome) entre otros. El cambio puede ser realizado manualmente o programado con cada dirección URL.

Lunascape proporciona también otras funcionalidades ya típicas en muchos navegadores como la navegación por pestañas, gestos de ratón, herramientas RSS, soporte de skins, autocompletado de formularios, barra de búsqueda, sistema de marcadores y la posibilidad de instalar complementos.

De esta forma, Lunascape se convierte en un navegador web apto tanto para el uso habitual en este tipo de aplicaciones como para comprobar la correcta visualización de una web en distintos navegadores sin tener que cambiar de aplicación.

Cada navegador muestra las páginas de forma distinta y, a veces, su uso no es sólo cuestión de gustos, sino también de compatibilidad. Para añadir algo nuevo a la contienda llega Lunascape, el primer navegador híbrido

Más allá de esta función, que hará las delicias de los diseñadores web y de quienes usan páginas compatibles con un solo navegador, Lunascape ofrece características propias de un navegador clásico, como pestañas, paneles, barra de marcadores, pieles y caja de búsqueda.

Lunascape también aporta pequeñas novedades. En una barra inferior, por ejemplo, hay botones para activar o desactivar la carga de vídeos, animaciones y scripts, solución rápida e inteligente. Otro botón añade a favoritos todas las pestañas cargadas.
