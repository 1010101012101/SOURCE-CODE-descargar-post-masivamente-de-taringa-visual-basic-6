Internet Download Manager (o con frecuencia poco llamado IDM) es un gestor de descargas shareware y la herramienta de acelerador de la gestión de descargas del navegador de Internet, sólo está disponible para el sistema de la explotación de Microsoft Windows.

IDM permite a los usuarios descargar automáticamente ciertos tipos de archivos, descargas divide en varias secuencias para una descarga más rápida y las listas de descargas recientes para facilitar el acceso a los archivos, también la búsqueda de espejos para aumentar la velocidad de descarga. Su algoritmo de segmentación dinámica incluye todo el proceso de descarga.
