Bsp TwoMap y WinBSP: http://www.mediafire.com/?33zw10qln16wnff

Dark Lunator: http://www.mediafire.com/?14gskj5catsg987

Este programa sirve para cambiar la iluminación del Mapa o Bien el Sky (Cielo), desconozco como se usa yo me manejo con el Valve Hammer Editor, pero dicen que es bastante útil.

Wad Extractor: http://www.mediafire.com/?re3fis9yizygj31

Esto funciona para sacar el .WAD de un mapa es decir EXTRAER las TEXTURAS de un mapa, la verdad que es muy útil y fácil de usar.

SpriteViewer: http://www.mediafire.com/?dbej2ojyb1dx97p

Esta Herramienta Sirve para VISUALIZAR los Sprites sin necesidad de compilar el mapa.

Batch Compiler: http://www.mediafire.com/?6gwaynoqfu3bk6c

Este programa se usa mucho para Compilar los mapas de .map a .bsp, es uno de los mas usados hoy en dia, es mas practico que el Valve Hammer Editor, y mas Veloz.

ResGen: http://www.mediafire.com/?lshxs4m8lctcigu

Este .exe se usa mucho para los que suben los mapas con .wad .wav y cosas que no trae el counter por Default genera un archivo llamado .res que viene acompañado con el .bsp hace que baje todo lo que necesita la gente para poder jugarlo y así no tire error.

Wally: http://www.mediafire.com/?wfxa4lyi7q9ta55
