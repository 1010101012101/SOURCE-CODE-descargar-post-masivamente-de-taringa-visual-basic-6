Envía mensajes anónimos entre los ordenadores y pcs de tu red, intranet de tu trabajo o colegio.
Realiza la misma función que el comando Net Send pero de forma anónima.

Este software es gratuito y se puede distribuir libremente.

INSTALACION

No requiere instalación descarga el archivo ZIP y leete las instrucciones


COMO USARLO

Introduce el nombre del remitente anónimo que verán los usuarios de tu intranet cuando reciban el mensaje.

Escribe el mensaje que desees que reciban los usuarios de la red.

Abre el archivo ListaIps.txt con cualquier editor de texto y escribe una IP de cada PC de destino por línea.

Cuando hagas click en enviar el mensaje se enviará a todos los ordenadores de tu intranet seleccionados. Y de forma anónima!!


NOTA: Para enviar obvio tendras que saber la IP de tu remitente, si no sabes como ver la IP de una pc te tendras que entrar a la pc de tu amigo o victima ir a conexiones de Red/ Propiedades/TCP/IP y veras esto:


Copia la direccion IP y agregala a la listaips.txt

No es necesario que el otro usuario tenga el programa. Salvo que el otro usuario quiera conversar.

Este Programa puede servir para chatear entre compañeros de oficina o colegio si no tienes internet o para no utilizar el messenger.
Tambien puedes utilizarlo para gastarle alguna broma al mas cabron de la oficina¡¡
