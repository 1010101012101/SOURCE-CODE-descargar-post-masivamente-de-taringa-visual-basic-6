WYSIWYG Web Builder - software para la creación de páginas Web, que crea automáticamente código HTML en un momento, sólo tiene que colocar los objetos, es decir, conocimientos de HTML de todos ustedes no es necesario. Sólo tienes que arrastrar y soltar objetos en cualquier lugar de la página. Después de esto sólo subir a tu servidor web, utilizando una función de la herramienta "Publish". Tiene prácticamente todas las posibilidades de MS Frontpage, pero contiene mucho más herramientas para trabajar con los objetos (imágenes, texto, tablas, etc.) Con WYSIWYG Web Builder página Web en Internet tendrá exactamente la misma forma que en un PC, independientemente del navegador.



Web Builder es un WYSIWYG (lo que se ve-es-lo-que-hay) programa utilizado para crear páginas web. WYSIWYG significa que el acabado página mostrará exactamente la forma en que fue diseñado. El programa genera HTML (HyperText Markup Language), mientras que las etiquetas apuntar y hacer clic en las funciones deseadas, se puede crear una página web sin saber HTML. Sólo tienes que arrastrar y soltar objetos a la página de posición 'en cualquier lugar' que desee y cuando haya terminado la publicará en su servidor web (utilizando la herramienta de construir en Publicar). Web Builder le da control total sobre el contenido y el diseño de su página web.
