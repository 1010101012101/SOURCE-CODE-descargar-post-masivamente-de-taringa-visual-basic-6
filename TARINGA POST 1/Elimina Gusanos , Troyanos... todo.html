Elimina troyanos, rootkits, virus y todo tipo de malware con Microsoft Standalone System Sweeper



Microsoft recientemente ha sacado la primer beta del programa Microsoft Standalone System Sweeper la cual se trata de una herramienta que protege nuestro sistema operativo en contra de virus, rootkits, troyanos y malware, cabe mencionar que esta aplicación está basada en Security Essentials la cual tenemos que descargar para luego instalarla en una memoria USB, CD o DVD para después al momento de reiniciar nuestro computadora empezar a eliminar el malware.

Microsoft Standalone System Sweeper pesa 250 MB, una vez descargada e instalada en USB,CD o DVD, podremos iniciar la búsqueda de rootkits, virus, troyanos y todo tipo de malware.

Microsoft Standalone System Sweeper es una aplicación freeware la cual puede ser instalada en los siguientes sistemas operativos: Windows XP, Windows Vista y Windows 7 en versiones de 32 y 64 bits

Sitio de http://connect.microsoft.com/systemsweeper
