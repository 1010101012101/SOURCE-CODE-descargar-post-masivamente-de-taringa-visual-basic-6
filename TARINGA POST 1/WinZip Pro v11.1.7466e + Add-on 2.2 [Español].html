WinZip 11.1 es una versión actualizada de nuestro más reciente lanzamiento —WinZip 11.0— compatible con Windows Vista. Además de las excelentes características introducidas en WinZip 11.0 (resumidas más adelante), incluye:

- Certificado para Windows Vista
- Barra de herramientas e iconos de programa estilo Vista
- Interfaz de usuario estilo Explorador de Vista
-Compatibilidad con Windows Vista y XP de 64 bits, así como también con Windows Vista, XP y 2000 de 32 bits
- Instalador de Software de Microsoft (MSI) para instalación personalizada

10.0. Aplicamos la interacción con nuestros usuarios para mejorar el Asistente para Tareas, y agregamos un número de mejoras en compresión y facilidad de uso, así como también la habilidad para abrir archivos .BZ2 y .RAR.

Tal como en WinZip 10.0, usted puede seleccionar WinZip 11.0 con características Standard o Pro, ambas ofrecen mejoras de compresión que hacen de WinZip 11.0 la versión más versátil de siempre. Nuestra nueva opción "mejor compresión" le permite que WinZip decida el mejor método de compresión basado en el tipo de archivo que se va a comprimir. Esto asegurará que usted maximice la compresión de cada archivo que agregue a su archivo Zip. WinZip 11.0 también le permite mejorar la compresión de archivos de música .wav sin comprometer la integridad del archivo o la calidad del sonido. Además, si usted necesita abrir un archivo .BZ2 o .RAR, WinZip puede ser ahora su herramienta de elección.

Si depende del Asistente para Tareas de WinZip Pro para configurar copias de seguridad de datos, archivado de documentos importantes o distribución de información, apreciará las mejoras disponibles en WinZip 11.0 Pro. Ahora puede configurar el Asistente para Tareas para enviar automáticamente el archivo de registro y/o archivo Zip resultante al finalizar la tarea; use nuestra nueva función de "búsqueda" para encontrar una carpeta específica en un servidor FTP y seleccione tareas personalizadas directamente desde el Asistente para Tareas y desde el menú de Quick Pick.
