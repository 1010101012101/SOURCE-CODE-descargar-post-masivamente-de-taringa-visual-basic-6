Autodesk ha lanzado 123D, una herramienta de modelado 3D gratuita, en fase Beta, para Windows. Este software permite crear objetos tridimensionales de una forma relativamente sencilla y está dirigida a aquellos usuarios que necesitan convertir una idea en un proyecto tridimensional que se pueda explorar, modificar y si se desea finalmente, convertir el objeto en real.
El producto no se limita a un programa, hay detrás un portal que aglutinará una comunidad de usuarios y empresas especializadas que podrán convertir nuestro diseño en un objeto real utilizando tecnología de vanguardia, como el corte por láser.
Primero os hablaré del programa y posteriormente de los servicios a los que se tiene acceso en la Web 123D App. Lo primero que hemos de hacer, como no, es descargar el programa del sitio oficial. No requiere ningún tipo de registro. Existe versión para 32 y 64 bits. El fichero de descarga (64 bits) es algo grande, 520 MB, lo que a priori puede dar idea de que se trata de un programa complejo.
