Windows Azure, la plataforma cloud computing de Microsoft capaz de soportar lenguajes de programación open source como PHP, Ruby On Rails y Python, aspira a atraer al mayor número de desarrolladores posible para que creen y alojen sus aplicaciones.

En este sentido han sido presentadas dos posibilidades para usar los servicios de la plataforma Windows Azure de forma gratuita. Los suscriptores de MSDN van a poder disfrutar de 8 meses de evaluación gratuitos, a partir de los cuales podrán pasar a una suscripción en la plataforma Windows Azure, mientras que el resto de usuarios disponen de un periodo de evaluacion gratuito hasta Octubre del 2011 disfrutando de una serie de horas, mensajes y almacenamiento.
