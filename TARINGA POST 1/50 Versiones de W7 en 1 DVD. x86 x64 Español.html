Versiones:

* E: Las ediciones E de Windows 7 incluyen las mismas funciones que las ediciones estandar de Windows 7, pero no incluye el navegador Internet Explorer 8. (Incluido en carpeta Programas de la ISO).
N: Las ediciones N de Windows 7 incluyen las mismas funciones que las ediciones estandar de Windows 7, pero no incluye el Reproductor de Windows Media ni tecnologias relacionadas como Windows Media Center o Windows DVD Maker. NO incorpora Internet Explorer 8. (Incluido en carpeta Programas de la ISO)
N1: Las ediciones N1 de Windows 7 incluyen las mismas funciones que las ediciones estandar de Windows 7, pero no incluye el Reproductor de Windows Media ni tecnologias relacionadas como Windows Media Center o Windows DVD Maker. SI incorpora Internet Explorer 8.
CORE: Las versiones CORE no es mas que un fondo plano y una consola (cmd.exe); muy parecido a nuestro WinPE, sin escritorios, iconos y efectos, entre otros.
