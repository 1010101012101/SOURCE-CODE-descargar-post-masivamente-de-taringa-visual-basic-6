Si instalaste Windows 7 Esto es importante para ti....

Microsoften un intento por acabar con la piratería de su nuevo y flamante sistema operativo Windows 7 va a lanzo una actualización de su herramienta WAT (Windows Activation Technologies), sustituta del archiconocido WGA, que detectará mas de 70 parches y activadores de windows 7. Así que los usuarios que hayan activado Windows 7 con algún activador o parche como Remove WTA, Chew WGA, 7LOADER, win 7 toolkit, 7 Emulation, activador-7, hazar, RTM Branch build activation windows 7 para piratear Windows 7 dejarán de tener su sistema activado y “original”.

Esta actualización es para todas las ediciones de Windows 7. La fecha prevista para esta actualización de la herramienta antipirateríade Microsoft para Windows 7 (WTA) es del 16 de febrero, vía actualizaciones automáticas esta actualización será tildada como importante.

Pasos para Evitar el mensaje de esta copia no es original de Windows 7 Compilación 7600 y activarlo. Hace relativamente poco ha existido una actualización de seguridad que detecta si el Windows 7 es Original, en caso de no serlo te aparecerá en pantalla Windows7 Compilación 7600 Esta copia de Windows no es Original. A continuación explicaremos paso a paso como evitar este mensaje y restaurarlo correctamente.

Desconectarse de Internet y Denegar Permisos

Lo primero que tenemos que hacer es Desconectarnos de Internet (Recomendamos quitar el Cable de Red para que se mantenga la configuración), Reiniciar y realizar los siguientes pasos. Si no lo hacemos desconectados de Internet, es posible que al día siguiente vuelva el Mensaje de Copia No Original de Windows.

1. Inicio - Programas- Accesios - Simbolo de Sistema (Botón Derecho) -
Ejecutar como Administrador
2. Ejecutar comando: takeown /f "%windir%System32Wat*"
3. Ejecutar comando: icacls "%windir%System32Wat*" /deny *S-1-1-0
Conectarse a Internet y Activarlo

Si hemos desconectado el cable de red, volver a ponerlo

1. Descargar Windows 7 Extreme Edition (Safe) 3.118 Descargar Download, o Download

Se nos activa el Botón "Activate", Clickamos y se Reinicia.
3. Ejecutar de nuevo, en la sección "Certificate" clickamos "Activate".
4. Reiniciar y ¡Ya esta!

Ya tenemos de nuevo el windows sin el mensaje de Copia no Original de Windows7. Lo que tenemos que hacer ahora es volver a poner el fondo de pantalla y reconfigurar el tema que tuvieramos puesto.
