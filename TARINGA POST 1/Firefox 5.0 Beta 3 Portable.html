Mozilla Firefox - más rápido, más seguro, más fácil de usar y deportivas una nueva mirada, esta última versión de Firefox establece un nuevo estándar para la innovación navegador web. Tres objetivos fundamentales para Firefox 4: Hacer el navegador más rápido, la construcción de soporte para más de gran alcance como los estándares web HTML 5, y dando a los usuarios un control total sobre sus identidades en línea en la web social. Eso último, en particular, es una buena noticia para nosotros, como a menudo hemos defendido que la identidad pertenece en el navegador. Mozilla ha estado experimentando con diferentes técnicas de gestión de identidad durante el último año o así, y es bueno ver a la empresa el tratamiento de la identidad del usuario como principal y quot;top 3 y quot; característica. La web todo es social ahora, y es el momento de la aplicación del usuario se inicia realmente en calidad de agente del usuario.





Mozilla Firefox se centra en la gestión mejorada de memoria, el rendimiento y la estabilidad, la mejora de XUL, y los nuevos componentes básicos tales como datos de la aplicación almacenados en SQLite. Mozilla Firefox podría romper algunas extensiones existentes y las aplicaciones basadas en la parte superior de Firefox, y que sin duda incluirá nuevas optimizaciones si se quiere construir en el popular navegador. Mozilla Firefox deben tener Javascript y características adicionales SVG si usted está en ese tipo de cosas.

Características de Firefox son:
- Fácil migración: Cambiando a Firefox nunca ha sido más fácil ahora que los datos de las importaciones de Firefox como favoritos, historia, configuración, cookies y contraseñas de Internet Explorer. Firefox también puede importar la configuración de Mozilla 1.x, Netscape y Opera.
- Nuevo tema por defecto: Un nuevo tema por defecto proporciona un aspecto nuevo y brillante para los usuarios de Windows y Linux.
- SmartUpdate: Una nueva característica notifica a los usuarios SmartUpdate de las nuevas versiones de Firefox para que el navegador está siempre al día.
- Ayuda: Un nuevo sistema de ayuda en línea hacen de este un navegador potente y amigable.
- Extensión / Theme Manager: Nueva Extensión y Gerentes Tema proporcionan una manera conveniente y segura para gestionar y actualizar los cientos de add-ons que configurar Firefox, aparte de otros navegadores.
Mozilla Firefox también incluye numerosas correcciones de errores y mejoras, incluida la mayor velocidad de carga de la página.
- Bloqueo de ventanas emergentes: los anuncios Detener molestos pop-up en sus pistas con Firefox incorporada en bloqueador de ventanas emergentes.
