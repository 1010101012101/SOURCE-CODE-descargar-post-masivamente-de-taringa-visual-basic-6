eXtreme Movie Manager Deluxe Edition 7.1.2.9




Descripción
Crea una completa y elegante base de datos de tus DivX
eXtreme Movie Manager es una muy buena utilidad para crear una rica base de datos con tus películas DivX, ya que ésta está especialmente preparada para este tipo de formato.
Detecta multitud de aspectos técnicos de las películas, el codec usado (tanto de audio como de vídeo), duración, tamaño, resolución de imagen, canales de sonido, número de CDs, etc.
Algunos de los muchos datos que te permite introducir son: título, intérpretes (con base de datos incluida) categoría, director, año, argumento, comentarios y valoración personal de la película entre otros muchos.
eXtreme Movie Manager también incluye un buscador integrado capaz de realizar búsquedas siguiendo distintos parámetros, una lista para apuntar a quién dejas tus CDs, bases de datos de actores y actrices, un generador de informes en HTML y varias posibilidades para importar datos.
