Windows Discovery Light Edition V4 es una versión mejorada de Windows XP SP3 desarrollada en base a las versiones anteriores obteniendo lo mejor de cada una de ellas.


Fue diseñada con los siguientes propositos:


Menor intervencion del usuario.


Obtencion de un maximo rendimiento.


Obtencion de la mejor compatibilidad.


Obtencion de un sistema mas estable y seguro.


Obtencion de diversos estilos visuales para un mejor agrado.


Consumo de menos recursos que un XP tradicional.








Caracteristicas:


Copia automatica de una carpeta con los drivers mas comunes a la unidad C:/


Las mejores opciones de rendimiento fueron configuradas y distintos componentes innecesarios fueron eliminados.


Principales cambios:


Se corrigieron algunos errores de la version anterior.


Tiempo de Instalacion optimizado.


Se mantuvo el menu ENVIAR A.


Se obtimiso mas el SO basandome en versiones mas recientes en su mayoria las qe estan en ingles.


Posee Drivers de Red, Sonido y Video, CPU, WLAN, LAN, etc.


Se integraron mas programas y nuevas versiones de los anteriores como tambien se aceleraron las conecciones de red e internet.


Durante la instalación, cuando falten 13 Minutos para terminar la instalación del SO, tardará un poco, alli instalará algunos programas necesarios para abrir aplicaciones, tales como, JAVA, .Net, Adobe Reader 9, Entre otros.


Para la Instalación de los Programas se Utilizó BOA Software.


Se reemplazaron los iconos del Escritorio, Mi PC Y Panel de Control.


Importante—–>No retirar el DVD hasta finalizar la Instalación.
