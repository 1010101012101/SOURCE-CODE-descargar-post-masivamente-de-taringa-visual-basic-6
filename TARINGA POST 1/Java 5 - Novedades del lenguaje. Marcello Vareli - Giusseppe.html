Java 5 - Novedades del lenguaje




PDF / Español / 13 mb / Marcello Vareli - Giusseppe Naccarato

Java 5, nombre en codigo "Tiger", es la ultima version de la plataforma desarrollo Java. Esta version ofrece muchas novedades que, en un sentido, revolucionaran nuestra manera de programar. Tipos genericos, enumeraciones, autoboxing son solo algunas de las novedades introducidas para hacer de Java un lenguaje todavia mas poderoso y flexible. Este libro les presentara en detalle las nuevas caracteristicas dando ejemplos y comparaciones con las versiones anteriores para comprender las diferencias y apreciar los beneficios.
