La mayoria de las computadoras y los principales fabricantes de equipos portatiles comenzaron a ofrecer Windows 7 con sus nuevos modelos de maquinas.

Lamentablemente, no proporcionan el disco de Windows 7 para la Instalacion y solo daban los discos de DVD con su compra. En su lugar, sacaron un disco llamado "disco de recuperacion" (eso si tienes suerte - de lo contrario tendra una particion de recuperacion en su lugar) con su equipo y dejarlo en eso.

Sin el disco de instalacion de Windows 7, sera imposible que usted pueda recuperar su instalacion de Windows 7. Afortunadamente, Microsoft parece haberse dado cuenta de este problema, y afortunadamente han hecho un disco de recuperacion para este fin.

Windows 7 recuperacion no puede ser utilizada para reinstalar o instalar Windows 7 y solo sirve como interfaz de Windows Para la recuperacion de su PC.

Windows 7 discos de recuperacion se puede utilizar para acceder a un menu de recuperacion del sistema, dandole opciones de utilizar Restaurar sistema, la linea completa de PC Backup, reparacion del sistema automatizado, y un simbolo del sistema para la recuperacion manual avanzado.
