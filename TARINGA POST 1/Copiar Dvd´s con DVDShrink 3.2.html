DVDShrink es un software para realizar copias de seguridad de discos DVD. Puedes usar dicho software junto con el programa de grabación de DVD que prefieras, para realizar copias de seguridad de tus discos DVD.
DVDShrink también graba los DVDs, si has instalado la última versión de Nero. Puedes descargarte una version demo de Nero aqui. Si ya posees un software de grabación alternativo y prefieres quedarte con él, todavía puedes usar DVDShrink. El resultado de DVDShrink se puede grabar como ficheros en tu disco duro, que pueden ser grabados con el software de tu elección.
DVDShrink es un software gratuito. No deberias pagar nunca por DVDShrink.
