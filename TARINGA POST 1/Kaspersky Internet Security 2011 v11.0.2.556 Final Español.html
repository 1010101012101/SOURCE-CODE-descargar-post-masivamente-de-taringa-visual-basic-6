Kaspersky Internet Security 2011 es una completa herramienta de protección de datos. La aplicación no sólo garantiza la protección anti-virus, sino también la protección contra el spam y los ataques de red, protege su computadora contra amenazas conocidas y desconocidas, los ataques de intrusos, además permite restringir el acceso de los usuarios a Internet. Cada tipo de amenaza es procesado por un componente de aplicación. Su nueva configuración flexible permite a los usuarios adaptar completamente Kaspersky Internet Security 2011 para sus necesidades específicas.

Kaspersky Internet Security 2011 ofrece una serie de nuevas y mejoradas características únicas, junto con las modernas tecnologías de protección para hacer frente a las últimas amenazas en línea, manten tu PC funcionando sin problemas y personaliza la protección acorde a tus necesidades:
Modo Seguro para aplicaciones, es un escritorio aislado donde puede iniciar aplicaciones aisladas sin dañar el sistema operativo.
System Watcher permite supervisar la actividad de las aplicaciones del sistema y proporciona información detallada sobre el resto de los componentes de protección. El componente puede deshacer los resultados de las acciones de una aplicación maliciosa cuando las detectan diferentes componentes de protección
Supervisor de vinculos url que advierte acerca de websites inseguras o infectadas
Filtrado Geográfico permite conceder o denegar el acceso a sitios Web en función de su pertenencia a determinados dominios
Modo de Juego para suspender alertas, actualizaciones y escaneos mientras juegas
Control de Aplicaciones para darte una imagen completa de los programas instalados en tu PC
Protección de Identidad Digital para darle una capa extra de protección a tus datos valiosos
Ingresa logins y contraseñas usando el seguro Teclado virtual
Sistema de Detección de urgencia para detener rápidamente las amenazas emergentes
Análisis en inactividad, se puede analizar el equipo en busca de virus mientras el usuario no está trabajando. El análisis se detendrá cuando vuelva a trabajar
Siguiente Generación en Protección proactiva ataca amenazas desconocidas desde el dia cero.
