Vectir es una aplicación de PC que permite controlar remotamente aplicaciones como Winamp, iTunes, Windows Media Player entre muchas otras cosas!.
Aca les dejo un screenshot para que se vallan dando una idea de como es!


http://www.vectir.com/


Basicamente es una aplicación cliente-servidor que te permite utilizar tu celular para controlar tu PC mediante WIFI o Bluetooth, tiene diferentes funciones predefinidas aunque también te permite crear nuevas funciones o perfiles. Por defecto viene con una amplia gama de actividades que puedes realizar desde tu teléfono, por ejemplo puedes controlar varios programas bastante populares como Windows Media Player, Winamp, iTunes y PowerPoint, Vectir te permitirá realizar todas las operaciones básicas de un reproductor multimedia como play, pause, stop etc. además de navegar por las playlist, e incluso actualizar el perfil por si aparecen nuevas funcionalidades. También te permite acceder a comandos del sistema como el volumen del audio del sistema, ingresar texto simulando un teclado, o la utilización del mouse, incluyendo dobles clik, clik derecho y scroll además de algunos comandos generales como Alt-Tab o apagar, reiniciar, o suspender el PC. Y por si todo esto fuera poco también puedes acceder a una especie de escritorio remoto donde podrás realizar zoom y manejar el zoom viendo todo en la pantalla de tu celular y del computador lo que lo convierte en una aplicación que puede controlar casi cualquier cosa de tu PC, realmente una herramienta muy útil.

Una amplia gama de celulares son compatibles, algunos de ellos son Nokia, Sony Ericsson, LG, Motorola, Samsung, Benq Siemens and Blackberry. Mientras tu dispositivo soporte Java y Bluetooth (o WIFI) vas a poder utilizarlo!

Los requerimientos del sistema para que te funcione este soft son:

* Microsoft Windows XP, Vista or 7 (32-bit y 64-bit)
* Microsoft .net framework 3.0
* 256MB RAM minimum, 512MB recommended
* 1GHz or higher CPU
* Bluetooth hardware if using Vectir for Bluetooth Remote Control
* Compatible IR hardware if using Vectir for IR Remote Control

La herramienta de escritorio (servidor) pueden descargarlo de acá: http://www.vectir.com/download/pc/vectir-setup-full.zip

El cliente para BlueTooth desde aca: http://www.vectir.com/download/mobile/mobile-bluetooth-remote-control/

Y el cliente para WIFI desde esta dirección: http://www.vectir.com/download/mobile/mobile-wifi-remote-control/

Yo lo instalé y anduvo perfecto en mi BlackBerry Curve 8520! Personalmente lo utilizo para manejar el WMP cuando estoy lejos de la pc y cuando estoy tirado en la cama y no me quiero levantar a cambiar el capitulo de la serie (es mi mouse inalambrico).



Instalación y configuración

1) Descomprimir el archivo descargado
2) Instalarlo (va a instalar el Microsoft .NET Framework 2 si no lo tenemos en nuestra PC)
3) Finalizar instalación

Como verán no hay complicaciones con esto

Luego tendrán que bajar la aplicación que quieran para su celu, ya sea la wifi o la bluetooth (a mi me anduvieron perfectas las dos), las instalamos en nuestro telefono movil y listo.

Vean que en el teléfono van a tener que poner la dirección IP de nuestro servidor en settings (la pc en la que está instalado el Vectir), esta la podemos averiguar facilmente llendo a la consola (Ejecutar -> cmd.exe) y luego ahi poner ipconfig. Donde dice IPAdress es su dirección.



Cada vez que quieran manejar su PC remotamente van a tener que tener abierto el programa en el server.
