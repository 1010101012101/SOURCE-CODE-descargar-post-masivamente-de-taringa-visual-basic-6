Descripción
Muchas empresas - necesitan algo más que de creación de PDF y la edición. Ellos necesitan la seguridad que garantice el cumplimiento normativo y la gestión empresarial de sus importantes documentos y archivos y para garantizar que los documentos no cambia después de ser aprobadas. Es esencial disponer de una robusta solución de PDF que se levanta a las demandas de las grandes empresas, pero lo suficientemente ligero para implementar fácilmente a miles de usuarios sin agotar los recursos. Estas empresas requieren Foxit PhantomPDF de negocios, sino que apoya las características adecuadas a un precio adecuado con la calidad y el apoyo que usted espera de una solución empresarial.

Business Ready PDF - Robusto y seguro
Foxit PhantomPDF de negocios se basa en las capacidades de PhantomPDF Standard y Express PhantomPDF. Permite la creación rápida y fácil de documentos PDF de aspecto profesional, poniendo de relieve y las capacidades de anotación, las capacidades avanzadas de edición, y la seguridad de gama alta para proteger la información sensible. Las principales funcionalidades de PhantomPDF de negocios:

XFA de datos en formularios - XFA (XML Formulario Arquitectura) formulario que permite aprovechar las formas XFA.
Alto Rendimiento - Hasta 3 veces más rápido de creación de PDF de más de 200 de los tipos de archivo más comunes de la oficina y convertir varios archivos a PDF en una sola operación.
Uno Haga clic en Creación de PDF - Crear documentos PDF con un solo clic del ratón desde las aplicaciones Microsoft ® Office, como Word ®, PowerPoint ® y Excel ®.

Advanced PDF Editor - Le permite modificar cualquier contenido de la página en los documentos PDF, seleccionar, insertar, modificar, borrar, rotar, copiar y pegar texto, imágenes, gráficos y matices.
Robusto Documento de Seguridad - Protección con contraseña, encriptación de certificados, y las herramientas de firma digital.

Redacción - Le permite eliminar de forma permanente (redactar) texto visible e imágenes de documentos PDF.
Documento de Certificación - Indica que se aprueba su contenido y le permite especificar los tipos de cambios que están permitidos para el PDF para mantener su certificación.

Microsoft ® Active Directory ® RMS Protector y Policy Manager - Se extiende los beneficios de controlar el uso de Microsoft Windows Server ® Active Directory Management Service derechos a todos sus documentos PDF y permite al autor el control de las características de cada documento PDF.

Formulario de diseño - Fácil de usar herramientas de diseño de formularios electrónicos para que su forma de trabajo de oficina más difícil. Permite crear o convertir archivos PDF estáticos en formularios de profesionales que buscan. Formulario de herramientas de importación de datos permiten que los datos se importan automáticamente en una forma clave de la reducción de la introducción manual y los errores de entrada.

Formulario de Acción de Creación - Herramientas que permiten a la costumbre acciones JavaScript para tener más opciones de personalización, puede conectarse a una base de datos con el fin de insertar nuevos datos, actualización de la información, y borrar las entradas de base de datos.

Reconocimiento óptico de caracteres - Soporte de OCR para escanear o archivos PDF con imágenes seleccionables y de búsqueda.

PDF / A validación - Verifique que el documento cumple con PDF/A-1a o PDF/A-1b para archivar a largo plazo.

Optimizador de PDF - Optimizar en formato PDF para reducir el tamaño del archivo.
Alto nivel de compresión de documentos escaneados.
