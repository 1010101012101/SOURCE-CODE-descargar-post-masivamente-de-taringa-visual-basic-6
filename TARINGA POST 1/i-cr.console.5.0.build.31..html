i-Catcher Console es un sistema de seguridad a través de vídeo e Internet, cuyos únicos requerimientos son un PC y una cámara web.


Una vez instalado, i-Catcher Console se encarga de detectar cualquier movimiento en pantalla, haciendo capturas de pantalla cuando se produzca este hecho. Entonces, el programa actuará según le hayas indicado previamente: enviarte las capturas por correo, enviarlas a una web, activar una alarma sonora, etc.

También es posible configurar el nivel de sensibilidad de i-Catcher Console a los movimientos que se produzcan. Realmente se trata de una utilidad perfecta para montar un sistema de vigilancia doméstico.

i-Catcher le permite construir un sistema completo de CCTV Digital, con detección de movimiento y alertas vía correo electrónico, SMS o la Web, así como funciones de reproducción, archivo, programación, y acceso Web, incluyendo reproducción remota en un navegador Web, escalando desde una sola PC con una cámara Web hasta una solución con múltiples locaciones cubriendo docenas de cámaras.

i-Catcher se integra con sistemas CCTV, cableado y cámaras ya existentes así como con Redes IP y Cámaras inalámbricas, asegurándole el mejor uso de cada recurso existente, y ofreciéndole flexibilidad completa en cualquier nuevo despliegue o actualización. Está disponible un acceso seguro a material en vivo y pregrabado desde la Consola, cualquier PC en la red local o por Internet.
