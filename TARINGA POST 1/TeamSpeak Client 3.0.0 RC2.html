TeamSpeak Client 3.0.0 RC2

TeamSpeak es flexible, de gran alcance, software escalable que permite a la gente a hablar unos con otros a través de Internet. TeamSpeak se compone de cliente y el software de servidor. El servidor actúa como anfitrión de múltiples conexiones de cliente, capaz de manejar miles de usuarios simultáneos.

La libreta de direcciones para facilitar la gestión de múltiples servidores de TeamSpeak
Crear múltiples canales y subcanales para sus usuarios
canales moderada para un mayor control al alojar reuniones de grupos grandes
Susurro funciones para que pueda hablar en privado con inidividuals, grupos de personas, o los usuarios de otros canales

Esta es la versión de 32 bits del cliente.
