Cambios
El instalador EZRound ahora es un instalador de Windows 7 cuenta y el programa EZRound ahora se manifiesta para Windows 7/Vista/2000/XP.
Cambio de ubicación predeterminada de los archivos creados por EZRound a "Mis documentos EZRound" para el cumplimiento de Vista, 2008 Server y Windows 7.
Añadida la opción de usuario para cambiar la ubicación de los archivos creados por EZRound.
Añadido soporte para la exportación de finas imágenes como archivos PNG con fondo transparente.
Añadida la opción para abrir el Explorador de Windows a la carpeta donde se guardan los archivos EZRound.
Actualización de archivos EZRound.ERT y agregó comillas simples alrededor del nombre de imagen de fondo para solucionar problema con IE8.
Añadida función para el divisor de HTML de la Gerente de Exportaciones para los usuarios NetObjects Fusion que se modifique la ruta de el código HTML generado sobre la base de una selección que el código se está utilizando en la página principal, o en otras páginas. La función está desactivada por defecto para los no-usuarios NetObjects Fusion.
Fija [BT # 3] de forma se distorsiona en la vista previa a veces (Ej: Nuevo proyecto, forma de rectángulo redondeado, vista previa).
Cambio de las fuentes de entrada de campo para un mejor soporte de fuentes ClearType.
