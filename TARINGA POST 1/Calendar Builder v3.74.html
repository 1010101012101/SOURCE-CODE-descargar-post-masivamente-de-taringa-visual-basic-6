Calendar Builder editor visual para crear sus propios calendarios. El tipo y el estilo que puede cambiar los acontecimientos en una amplia gama: desde una semana a 60 meses en la hoja. Para la mayoría de calendario le permite añadir imágenes, texto, bordes, fondos, rellenar el gradiente del campo, etc El resultado es una imagen en formato BMP o JPG, puede imprimir o guardar.







Calendar Builder es una herramienta WYSIWYG de diseño visual para crear sus propios calendarios de encargo, con colores, imágenes, iconos de eventos, áreas de texto personalizado, bordes, fondos, textos y más degradado. Es compatible con eventos recurrentes, como cumpleaños, aniversarios, días festivos (lista incluida), y usted puede elegir entre más de una docena de diferentes tipos de calendario. Cada tipo de calendario se pueden personalizar y guardar como una plantilla para su uso posterior. El calendario final se puede imprimir o guardar como archivo BMP o JPG.
