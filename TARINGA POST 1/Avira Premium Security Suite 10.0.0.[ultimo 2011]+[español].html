Avira Premium Security Suite 10.0.0.6[ultimo 2011]+[español]







Avira Premium Security Suite 10.0.0.621 fue diseñado para proteger al usuario y tiene un FireWall adicional, así como un AntiSpam y una función de copia de seguridad Además de la protección antivirus galardonada.

En una instalación definido por el usuario o una instalación de modificación, los siguientes módulos de instalación pueden ser seleccionados, añadidos o eliminados.

Prima Security Suite
Este módulo contiene todos los componentes necesarios para la instalación exitosa de Avira Premium Security Suite.

Guardia de AntiVir
La Guardia de AntiVir se ejecuta en segundo plano. Monitorea y reparaciones, donde sea necesario, archivos durante las operaciones, como abrir, escribir y copiar en modo real. Cuando un usuario realiza una operación de archivo (por ejemplo, cargar documento, ejecutar, copiar), Avira Premium Security Suite analiza automáticamente el archivo. Cambiar el nombre de un archivo, no activa un análisis por guardia de AntiVir.

AntiVir MailGuard
MailGuard es la interfaz entre el equipo y el servidor de correo electrónico desde la que tu programa de correo electrónico (cliente de correo) descarga los correos electrónicos. MailGuard está conectado como un llamado proxy entre el programa de correo electrónico y el servidor de correo electrónico. Todos los correos electrónicos entrantes se enrutan a través de este proxy, analizan en busca de virus y programas no deseados y reenviados a su programa de correo electrónico.Dependiendo de la configuración, el programa automáticamente los procesos afectados de los mensajes de correo electrónico o pide al usuario una acción en concreto.Además, la MailGuard puede confiable protegerle contra mensajes de correo electrónico de spam.

AntiVir WebGuard
Cuando navega por internet, estás utilizando el navegador web a datos de solicitud desde un servidor web.Normalmente, se moverán los datos transferidos desde el servidor de web (archivos HTML, secuencia de comandos y archivos de imágenes, archivos de Flash, secuencias de vídeo y música, etc.) directamente en la caché del navegador para mostrar en el Explorador de web, lo que significa que un análisis en tiempo real como realizada por la Guardia de AntiVir no es posible. Esto podría permitir virus y programas no deseados para tener acceso a su sistema informático.WebGuard es lo que se conoce como un proxy HTTP que supervisa los puertos utilizados para la transferencia de datos (80, 8080, 3128) y analiza los datos transferidos de virus y programas no deseados.Dependiendo de la configuración, el programa puede procesar de forma automática los archivos afectados o solicitar al usuario una acción específica.

AVIRA Firewall
AVIRA Firewall controla la comunicación hacia y desde el equipo.Se permite o deniega las comunicaciones basadas en políticas de seguridad.

Detección de rootkit
La detección de Rootkit comprueba si el software ya está instalado en el equipo que ya no puede ser detectado con los métodos convencionales de protección frente a malware después de penetrar en el sistema informático.

Extensión de shell
La extensión de shell de Suite de seguridad de Avira Premium genera un archivos de análisis seleccionado de entrada con AntiVir en el menú contextual del Explorador de Windows (botón derecho del ratón).Con esta entrada directa puede analizar archivos o directorios.
