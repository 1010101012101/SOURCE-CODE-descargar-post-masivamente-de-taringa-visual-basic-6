Image for Windows es una utilidad para realizar copias de seguridad del disco del sistema de archivos. La copia de seguridad puede ser retirado de cualquier unidad de disco duro, incluyendo unidades de red. El programa cuenta con acceso directo a los sistemas de archivos FAT, FAT32, NTFS, así como USB2, IEEE1394, ATAPI o ASPI (CD-R/RW o DVD + RW / + R /-R /-RW). Usted puede escribir copias de seguridad en CD o DVD, crear un disco de arranque para la recuperación automática. El programa también es compatible con una interfaz de RAID.
