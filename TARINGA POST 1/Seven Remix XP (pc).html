Si aún trabajas con Windows XP, pero te gustaría que tenga el aspecto de Windows 7; Seven Remix XP podría ser de ayuda. Seven Remix XP es una aplicación que fue diseñada para mejorar la GUI de Windows XP. El programa de instalación reemplazará algunos archivos del sistema, archivos que contienen íconos, imágenes, animaciones que serán extraídas de Windows 7, a fin de que XP tenga la apariencia de Windows 7 de forma practica y muy ligero.

Este paquete trabaja con los siguientes sistemas operativos:

* Windows XP (Home, Pro, RTM, SP1, SP2, SP3, Corporate)
* Windows 2003 (Standard, Enterprise, Datacenter, Web, RTM, SP1)
* Windows Media Center Edition (2003, 2004, 2005)
* Windows XP Tablet PC Edition
* Todas las versiones lingüísticas
* No funciona con versiones de Windows 64 bits

El programa instalador sustituye archivos de sistema, hace cambios en el registro e instala los skins y archivos de sonidos, estilos, etc. Además, hace una copia de seguridad de los archivos, por si luego queremos recuperarlos. Se puede desinstalar el paquete en cualquier momento que uno lo desee. Les sugiero instalarlo tras una instalacion de SO limpia (reciente) veran los resultados.
